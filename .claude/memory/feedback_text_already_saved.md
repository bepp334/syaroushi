---
name: テキスト保存済みの場合は問題作成に直進する
description: テキストが既にファイルに保存されている場合、貼り付けを求めず即座に問題作成に進む
type: feedback
---

テキストフォルダに該当ファイルが既に存在する場合は「テキストを貼り付けてください」と聞かずに、そのままファイルを読み込んで問題作成に進む。

**Why:** ユーザーが事前にテキストを保存している場合、再度貼り付けを求めるのは無駄な手間。ファイルの存在確認→即問題作成がスムーズな流れ。

**How to apply:** 「次へ」「問題作成」「〇〇の問題を作って」と言われたら、まずテキストフォルダを確認し、ファイルがあれば読み込んで問題作成に進む。ファイルがない場合のみ貼り付けを求める。
