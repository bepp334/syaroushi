# 社労士問題出力・解説ガイド

社労士試験の問題を出題・解説するときの形式ルール。

---

## トリガー

以下の発言パターンで発動する：

- 「問題作って」「問題演習」「問題出して」
- 「解説して」「説明して」（採点後）
- 「claude.mdの方式で」「正しい形式で」
- 「エピソードで解説して」「具体的に解説して」「ストーリーで解説して」

---

## 問題出力のルール

### 択一式の出力形式

チャット上で問題を出すときも、以下の本番形式をそのまま出力する。要約・省略・箇条書き化は絶対にしない。

```
「〇〇法第△条に関する次の記述のうち、正しいもの（誤っているもの）はどれか。」

A. [主語から始まる完結した1文（1〜3行）]

B. [主語から始まる完結した1文（1〜3行）]

C. [主語から始まる完結した1文（1〜3行）]

D. [主語から始まる完結した1文（1〜3行）]

E. [主語から始まる完結した1文（1〜3行）]
```

**NG例：**
- 選択肢が「A. 合意すれば可」など断片的な文
- 「→」で省略する形式
- リード文が短すぎて何を問うているか分からない形式

---

## 解説のルール

### 採点後の解説（択一式）

正解・不正解にかかわらず、全問題に対して同じ深さ・同じフォーマットで解説する。「正解だったので省略」は絶対にしない。

```
【問題文】
（リード文＋A〜Eの全選択肢を省略せず一字一句そのまま掲載する。要約・短縮・「〜」での省略は絶対にしない）

【この問題を解くために必要な前提知識】
→ 「そもそも何を知っていなければならないか」を最初に説明
→ 専門用語は「つまり〇〇ということ」と噛み砕く
→ 身近な例・ストーリーで直感的に理解させる
→ 「なぜこのルールがあるのか」制度の背景・目的も説明

【問題文の読み解き方】
→ A〜E各選択肢の正誤と理由を1文ずつ解説
→ 「〇〇という言葉がポイント」など具体的にどこを見れば判断できるか示す

【正解】〇

【試験で狙われるポイント！】
→ ひっかけワード・覚え方・頻出パターン
```

### 採点後の解説（選択式）

```
【問題文】
（穴埋め文＋選択肢を省略せず全部掲載する）

【この問題を解くために必要な前提知識】
→ 空欄に入る概念の意味・背景・なぜその言葉なのかを説明

| 空欄 | 正解 | 誤りやすい選択肢 | なぜ誤りか |
|------|------|----------------|----------|
|  A   | 〇〇 |     △△        |  理由    |

【セットで覚える！】
→ 複数の空欄がある場合、セットで覚えるポイントをまとめる

【試験で狙われるポイント！】
```

## エピソード解説のルール

ユーザーから「エピソードで」「具体的に」「ストーリーで」「中学生にもわかるように」と言われたとき、または通常解説の後に深掘りを求められたときに使う。

### 形式

A〜E各選択肢を**すべて**、以下の構成で解説する。正解・不正解にかかわらず省略しない。

```
**[選択肢記号]「選択肢の核心を一言で」→ ✅ or ✗**

> [登場人物]が[状況]に置かれたエピソード（2〜4文）
>
> → [法律上の結論と理由を1文で]
```

### エピソード作成のルール

- **登場人物は固定**：田中さん（会社員）・田中さんの上司・田中さんの友人・田中さんの祖父など、シリーズで同じ人物を使い回すと記憶に定着しやすい
- **場面は日常生活に近い設定**：工場・コンビニ・農園・病院・役所など身近な職場
- **論点のひっかけポイントをエピソードの中に自然に埋め込む**：「いかなる時点から切り取っても」「当然に」「〜しなければならない」などのキーワードを会話や状況の中で体験させる
- **正しい選択肢のエピソードは「なぜそのルールがあるのか」の背景も入れる**
- **誤りの選択肢のエピソードは「どこが間違いか」がオチになるように組み立てる**

### エピソード解説の末尾

全選択肢の解説が終わったら必ず以下を添える：

```
---
**【このエピソードで覚えるポイント】**
→ ひっかけワード・セットで覚えること・頻出パターンを箇条書きで3点以内に絞る
```

---

## 採点後の結果記録（自動）

解説が終わったら、必ず該当の問題ファイルの【結果記録】欄をEditツールで更新する。

```
## 【結果記録】
解いた日：YYYY-MM-DD
正解数：X／Y問
間違えた問題番号：（番号をカンマ区切り）
コメント：（主な間違いパターンを一言で）
```

省略・後回し禁止。解説を書いた直後に必ず記録する。
