# 問題作成プロンプト

テキストから問題を作成するときにこのプロンプトをAIに渡してください。

---

## 基本の問題作成プロンプト

以下のテキストから社労士試験対策の問題を作成してください。

**作成ルール：**
1. **択一式（5択）**: 3〜5問

   **問題文の形式（本番準拠）**：
   ```
   「〇〇法第△条に関する次の記述のうち、正しいもの（誤っているもの）はどれか。」

   A. [主語＋状況説明＋結論]の完結した文（1〜3行）
   B. ...
   ```
   - 問題文（リード文）は「〇〇に関する次の記述のうち〜」の形にする
   - 各選択肢は必ず「主語から始まる完結した1文」にする（断片的な語句はNG）
   - 選択肢を見ただけで何を聞かれているか分かる形にする
   - 正解は1つ
   - 紛らわしい選択肢を含める（数字・期間の微妙な違い、「のみ」「一部」「全部」の差など）
   - 「正しいものはどれか」「誤っているものはどれか」を混在

2. **選択式（空欄補充）**: 2〜3問
   - 本番の選択式を意識した穴埋め
   - 空欄は数字・期間・固有名詞など重要キーワード

3. **問題の後に以下を付ける：**
   - 【解答】
   - 【解説】：なぜその答えか（根拠条文のイメージ）
   - 【覚え方】：語呂合わせやポイント
   - 【出題傾向】：過去問でよく出るパターン

4. **難易度バランス**: 基礎50% / 標準30% / 応用20%
