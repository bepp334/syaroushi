## クイックスタート

### テキストを読んで理解したいとき
1. テキストをコピペしてAIに貼り付ける
2. 「プロンプト集/解説プロンプト.md のスタイルで解説して」と伝える

### テキストから問題を作りたいとき
1. テキストをコピペしてAIに貼り付ける
2. 「問題を作って」と伝える（プロンプト集/問題作成プロンプト.md が自動適用）

### 間違いを記録したいとき
- 「〇〇を間違えた。記録して」と伝える
- AIが `間違いデータベース.md` と `科目/[科目]/間違いノート.md` に記録

### 今日何を勉強すべか確認したいとき
- 「今日何を勉強すればいい？」と聞く
- AIが `学習管理/合格戦略.md` を参照して提案

### 弱点を分析したいとき
- 「弱点を分析して」と伝える
- AIが `間違いデータベース.md` を分析して報告

## 学習サイクル（基本の流れ）

```
テキスト読む
    ↓
AIに貼り付けて理解（解説プロンプト）
    ↓
AIに問題を作成してもらう
    ↓
問題を解く
    ↓
間違えた問題を記録
    ↓
翌日・翌週に復習キューから再出題
    ↓
間違いが減ったら次の範囲へ
```
