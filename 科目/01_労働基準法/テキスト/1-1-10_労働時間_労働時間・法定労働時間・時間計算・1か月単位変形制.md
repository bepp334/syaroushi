# 1-1-10 労働時間・休憩・休日（2）労働時間・法定労働時間・時間計算・1か月単位の変形労働時間制

---

## 1-1 労働時間

### 1-1-1 労働時間の定義 ★★★★★

**労働時間とは：** 労働者が**使用者の指揮命令下に置かれている時間**
→ 必ずしも実際に精神・肉体を活動させていることを要件としない
→ 労働契約・就業規則・労働協約等の定めに関係なく**客観的に定まる**（三菱重工業長崎造船所事件・最判平12）

| 労働時間になる | 労働時間にならない |
|--------------|-----------------|
| **手待時間**（貨物待ちのトラック運転手の仮眠等） | 一般健康診断の受診時間 |
| 来客当番（休憩中でも事務所待機） | 出席の強制のない教育・研修時間 |
| **特殊健康診断**の受診時間 | 任意に出勤して従事した消火作業時間 |
| 安全・衛生委員会の会議時間 | |
| **出席の強制のある**教育・研修時間 | |
| （原則）訪問介護労働者の移動時間 | |

⚠️ 訪問介護の移動時間：使用者が移動を命じ、自由利用が保障されていない場合は労働時間に該当

**不活動時間（仮眠時間等）の判断基準（大星ビル管理事件・最判平14）：**
実作業をしていない時間でも、**労働からの解放が保障されていない**場合は労働時間に該当

**マンション住込み管理員（大林ファシリティーズ事件・最判平19）：**
所定労働時間外でも管理員室隣で待機を余儀なくされている時間 → 労働時間
一方、通院・犬の散歩などの私的な時間 → 労働時間ではない

---

### 使用者の労働時間管理義務 ★★★★

- 使用者は労働者の始業・終業時刻を確認し記録する義務
- 原則的方法：①使用者が自ら現認 または ②タイムカード・ICカード等の客観的記録
- 管理対象外：法41条該当者・みなし労働時間制適用者

---

### 1-1-2 法定労働時間 ★★★★★

**原則（法32条）：**

```
1週間：休憩時間を除き40時間以内
1日：休憩時間を除き8時間以内
違反：6か月以下の拘禁または30万円以下の罰金（法119条1号）
```

**「1週間」の定義：** 就業規則等に別段の定めがない限り、日曜日〜土曜日の暦週
**「1日」の定義：** 原則として暦日（午前0時〜午後12時）
→ ただし継続勤務が2暦日にわたる場合は「1勤務」として扱い、始業時刻の属する日の労働とする

---

### 法定労働時間の特例 ★★★★★

以下の**特例事業**のうち**常時10人未満**の労働者を使用するものは、1週44時間・1日8時間まで可：

| 特例事業 | 注意点 |
|---------|--------|
| 商業 | |
| 映画・演劇業 | **映画の製作の事業は除く**（製作は40時間） |
| 保健衛生業 | |
| 接客娯楽業 | |

⚠️ 「10人未満」が要件（10人以上はNG）
⚠️ 1週「44時間」・1日「8時間」（48時間・10時間ではない）
⚠️ 10人未満の算定：週2日勤務の者など継続的に働く者も含む

### 1-1-3 時間計算

#### 事業場を異にする場合（異事業通算制）★★★★★

**法38条1項：** 労働時間は、事業場を異にする場合においても**通算する**
→「事業場を異にする場合」には**事業主を異にする場合も含む**（副業・兼業の場合も通算）

**通算されない者：**
- フリーランス・独立・顧問等（そもそも労基法不適用）
- 農業・管理監督者・高度プロフェッショナル等（労働時間規制不適用）

**通算して適用される規定：**
- 法定労働時間（法32条・法40条）
- 時間外労働の上限（月100時間未満・複数月平均80時間以内）

**通算されない規定（事業場ごとに管理）：**
- 36協定の延長時間の限度・年上限
- 36協定に定めた延長時間の範囲内かどうか
- 休憩・休日・年次有給休暇

#### 副業・兼業と割増賃金の負担

**原則：後から契約した使用者が割増賃金を負担**
（契約締結時に他社での労働を確認できるため）

**手順：**
1. 契約締結の**先後**の順に所定労働時間を通算
2. 所定外労働の**発生順**に所定外労働時間を通算
3. 法定労働時間を超えた部分のうち、**自ら労働させた時間**について割増賃金を支払う

**具体例：**
```
A社（先）8時間労働 → A社は法定内のため割増なし
その後B社（後）4時間労働 → すべて法定時間外
→ B社が割増賃金を支払う（36協定も必要）
```

#### 坑内労働 ★★★★★

坑内労働：**坑口に入った時刻〜坑口を出た時刻**を、**休憩時間を含め**労働時間とみなす
→ 休憩の一斉付与の原則・自由利用の原則は適用されない

## 1-2 変形労働時間制

### 1-2-1 変形労働時間制の概要

**種類：**
| 制度 | 条文 | 届出 | 有効期間 |
|------|------|------|---------|
| 1か月単位の変形 | 法32条の2 | 要 | 要 |
| フレックスタイム制（1か月以内） | 法32条の3 | **不要** | 不要 |
| フレックスタイム制（1か月超） | 法32条の3 | 要 | 要 |
| 1年単位の変形 | 法32条の4 | 要 | 要 |
| 1週間単位の非定型的変形 | 法32条の5 | 要 | 不要 |

**育児等を行う者への配慮（則12条の6）：**
1か月単位・1年単位・1週間単位の変形労働時間制に適用
⚠️ フレックスタイム制は対象外

---

### 1-2-2 1か月単位の変形労働時間制 ★★★★★

**概要：** 1か月以内の期間を平均して1週40時間（特例事業は44時間）を超えなければ、特定の日・週に8時間・40時間を超えて労働させることができる

**採用の要件：**
```
「労使協定」または「就業規則その他これに準ずるもの」に定める
（どちらを使うかは使用者が決定）
```

⚠️ ただし労使協定の免罰効果のみでは労働者に従う義務は生じない
→ 労使協定で採用する場合も、就業規則への規定が必要

**定める事項：**
1. 変形期間（1か月以内の一定期間）
2. 変形期間の起算日
3. 各日・各週の労働時間
4. 1週平均が法定労働時間を超えない旨

**届出：**
- 労使協定による場合：所轄労働基準監督署長への届出**必須**（違反：30万円以下の罰金）
- 就業規則による場合：10人以上の事業場では就業規則の届出（10人未満は届出不要）

**変形期間の法定労働時間の総枠：**
```
法定労働時間の総枠 = 週法定労働時間（40h or 44h）× 変形期間の暦日数 ÷ 7

例：1か月（30日）の場合 = 40 × 30 ÷ 7 = 171.4時間
例：1か月（31日）の場合 = 40 × 31 ÷ 7 = 177.1時間
例：4週間の場合 = 40 × 28 ÷ 7 = 160時間
```
⚠️ 「労働日数÷7」ではなく「**暦日数**÷7」

**労働時間の特定（重要）：**
各日・各週の労働時間を**事前に具体的に特定**する必要がある
→「業務の都合により変形期間を平均して1週40時間の範囲内で就業させることがある」という漠然とした定めでは不可（大星ビル管理事件・最判平14）

**1か月単位の変形労働時間制と事業場規模：**
```
常時10人以上 → 労使協定（届出要）または就業規則（届出要）
常時10人未満 → 労使協定（届出要）またはその他これに準ずるもの（届出不要）
```
⚠️ 10人以上の事業場では「その他これに準ずるもの」は使えない（就業規則が必要）

**特例：列車等に乗務する予備勤務者（則26条）**
労使協定等の定めがなくても、1か月平均40時間以内であれば週40時間・日8時間超の労働が可能
（対象：列車・気動車・電車の乗務員の予備勤務者のみ。航空機は対象外）

**勤務ダイヤによる場合：**
業務の実態から月ごとに勤務割を作成する必要があるときは、各直勤務の始業終業時刻・組み合わせ・作成手続・周知方法を就業規則等に定めておき、各日の勤務割を**変形期間の開始前までに具体的に特定**することで足りる
