# 1-1-9 労働時間・休憩・休日（1）法41条該当者・高度プロフェッショナル制度・労使協定

---

## 1-1 労働時間・休憩・休日の適用除外

### 1-1-1 適用除外者（法41条該当者）★★★★★

法41条該当者には、**第4章（労働時間・休憩・休日・年次有給休暇）・第6章（年少者）・第6章の2（妊産婦等）で定める労働時間・休憩・休日に関する規定**は適用されない。

| 類型 | 要件 | 重要度 |
|------|------|--------|
| ① 農業・水産業・畜産業に従事する者 | 別表第1第6号（林業除く）または第7号 | ★★★★★ |
| ② 監督・管理の地位にある者 または 機密の事務を取り扱う者 | 事業の種類を問わない | ★★★★★ |
| ③ 監視または断続的労働に従事する者 | **所轄労働基準監督署長の許可**が必要（届出では足りない） | ★★★ |

**⚠️ 林業は除外されない！**
→ 農業・水産業・畜産業は法41条該当、**林業は非該当**（比較的計画的な作業で自然条件の影響が低いため）

**機密の事務を取り扱う者とは：**
秘書その他、職務が経営者または監督・管理の地位にある者の活動と一体不可分で、厳格な労働時間管理になじまない者（昭22.9.13発基17号）

**監視に従事する者：** 監視を本来の業務とし、常態として身体または精神的緊張の少ないもの
**断続的労働に従事する者：** 休憩時間は少ないが手待時間が多い者

---

### 宿日直勤務 ★★★★★

- 断続的労働の一態様として**宿直・日直の勤務（断続的な業務）**も含む
- 所轄労働基準監督署長の許可が必要
- 常態としてほとんど労働をする必要のない勤務（定時的巡視・緊急文書電話の収受・非常事態への待機等）のみ許可
- 宿直手当・日直手当の最低額：**同種労働者の賃金の1人1日平均額の3分の1**を下回らないこと（「2分の1」ではない）

**宿直中に通常業務を行わせたとき：**
→ 宿直手当に加え、当該時間に対して**時間外労働として割増賃金**を支払わなければならない

**医師・看護師などの宿直業務：**
① 通常の勤務時間の拘束から完全に解放された後のものであること
② 夜間の業務は軽度または短時間の業務に限ること
③ 夜間に十分な睡眠がとりうること
④ 一般の宿直許可の条件を充たすこと
→ 上記すべてを満たす場合のみ許可

---

### 1-1-2 適用除外の範囲 ★★★★★

法41条によって**適用除外されないもの（引き続き適用されるもの）：**

| 規定 | 適用 | ポイント |
|------|------|---------|
| 年次有給休暇 | **適用される** ★★★★★ | 除外なし |
| 深夜業の割増賃金 | **適用される** ★★★★★ | 選択注意 |
| 時間外・休日労働の割増賃金 | 適用されない | 労働時間規制の一部 |

**なぜ深夜業は適用されるか（ことぶき事件・最判平21.12.18）：**
- 時間外・休日の割増賃金 = **労働時間の「長さ」**に着目
- 深夜業の割増賃金 = **「時間帯」**に着目（趣旨目的が異なる）
- → 法41条の「労働時間・休憩・休日に関する規定」には深夜業規制は含まれない

⚠️ 管理監督者でも**深夜割増賃金の請求はできる**。
ただし、労働協約・就業規則等で深夜割増賃金を含めて所定賃金が定められていることが明らかな場合は別途支払不要。

### 1-1-3 管理監督者性の判断基準 ★

**一般的定義：** 部長・工場長など、労働条件の決定その他労務管理について経営者と一体的な立場にある者
→ **名称にとらわれず、実態に即して判断**

**3つの判断軸：**
1. 職務内容・責任と権限
2. 勤務態様
3. 賃金等の待遇（その地位にふさわしい待遇がなされているか）

※ 一般労働者より優遇されていても、実態のない役付者は管理監督者に含まれない

**名ばかり管理職（多店舗展開の小売・飲食業の店舗）の判断要素（平20.9.9基発0909001号）：**

| 要素 | 管理監督者性を否定する重要な要素 |
|------|-------------------------------|
| 採用に実質的な責任と権限がない | 否定の重要な要素 |
| 解雇に関与しない | 否定の重要な要素 |
| 人事考課に関与しない | 否定の重要な要素 |
| 勤務割表作成・残業命令の権限がない | 否定の重要な要素 |
| 遅刻・早退で減給等の不利益がある | 否定の重要な要素 |
| 労働時間の裁量がほとんどない | 否定の**補強**要素 |
| 部下と同様の勤務態様が大半 | 否定の**補強**要素 |
| 割増賃金適用除外を考慮すると待遇が不十分 | 否定の**補強**要素 |
| 時間単価がアルバイト・パート以下 | 否定の**重要な**要素 |
| 時間単価が最低賃金額未満 | 否定の**極めて重要な**要素 |

## 1-1-4 高度プロフェッショナル制度（法41条の2）

**概要：**
年収要件を満たし、職務の範囲が明確で高度の専門的知識等を有する労働者を対象に、
労働者本人の同意・労使委員会の決議等を要件として、
**労働時間・休憩・休日・深夜の割増賃金**に係る規定を適用除外とする制度。

⚠️ **法41条との違い：深夜業の割増賃金も適用除外**（法41条は深夜業は適用される）

### 対象業務（則34条の2第3項）
1. 金融商品の開発業務
2. 投資判断に基づく資産運用業務（ディーリング業務等）
3. 有価証券の価値分析・評価・助言業務（アナリスト業務）
4. 顧客事業の重要事項についての調査・分析・助言業務（コンサルタント業務）
5. 新たな技術・商品・役務の研究開発業務

### 対象労働者の要件
1. 書面等の合意に基づき**職務が明確に定められている**こと
2. 見込み年収が**1,075万円以上**（基準年間平均給与額の3倍を相当程度上回る水準）

### 採用の要件
| 要件 | 内容 |
|------|------|
| 労使委員会の設置 | 使用者＋労働者代表を構成員とするもの |
| 決議 | 委員の**5分の4以上**の多数による議決 |
| 届出 | 所轄労働基準監督署長へ届出（**効力の発生要件**） |
| 本人同意 | 書面等による対象労働者の同意（撤回可能） |

### 3つの必須措置（いずれかを欠くと制度適用不可）
| 措置 | 内容 |
|------|------|
| ① 健康管理時間の把握 | 事業場内在社時間＋事業場外労働時間の合計（客観的方法） |
| ② 休日確保措置 | **1年104日以上**かつ**4週4日以上**の休日を与える（完全週休2日制が前提：52週×2日） |
| ③ 選択的措置 | 下記のいずれか1つ |

**選択的措置（いずれか1つ）：**
- 勤務間インターバル（11時間以上）＋深夜業**月4回以内**
- 健康管理時間の上限（**月100時間以内**または**3か月240時間以内**）
- 連続休日（**1年1回以上の継続2週間**、または労働者請求の場合1年2回以上の継続1週間）
- 臨時健康診断（健康管理時間が月80時間超または本人申出の場合）

### その他
- 健康・福祉確保措置（6項目から決議で選択）
- 報告義務：決議有効期間の始期から**6か月以内ごと**に監督署長へ報告
- 記録の保存：有効期間中および満了後**5年間（当分の間3年間）**

## 1-2 労使協定

### 1-2-1 労使協定 ★★★★★

**定義：** 当該事業場の過半数労働組合（ない場合は過半数代表者）との**書面による協定**

**締結単位：** 事業場ごと（ただし本社一括届出の特例あり）

### 過半数代表者の要件 ★★★★★

1. **法41条2号の監督・管理の地位にある者でないこと**
2. 労使協定の締結当事者を選出することを**明らかにして**実施される投票・挙手等の手続により選出された者であって、**使用者の意向によって選出されたものでないこと**

⚠️ 管理監督者は労働者数の算定には含まれるが、**過半数代表者にはなれない**
⚠️ 管理監督者のみの事業場では、2の要件のみを満たせばよい

### 労使協定の効力

**免罰効果：** 協定の定めによって労働させても労基法に違反しないという効果
→ 労働者の民事上の義務は労使協定から直接生じない（労働協約・就業規則等の根拠が別途必要）

**例外：年次有給休暇の計画的付与に係る労使協定**
→ 免罰効果＋**民事的効力**（個々の労働者の時季指定権・使用者の時季変更権を制約）を有する
→ 労働協約や就業規則等の根拠は不要

**効力の及ぶ範囲：** 組合員でない者や反対している者にも及ぶ

**要件の喪失：** 締結後に過半数要件を満たさなくなっても、**締結時に要件を満たしていれば効力を失わない**（成立要件であって存続要件ではない）

### 届出・有効期間のまとめ ★★★★★

| 労使協定 | 届出 | 有効期間の定め |
|---------|------|--------------|
| 任意貯金 | 要（罰則なし） | 不要 |
| 賃金の一部控除 | **不要** | 不要 |
| 1か月単位の変形労働時間制 | 要 | 要 |
| フレックスタイム制（清算期間1か月以内） | **不要** | 不要 |
| フレックスタイム制（清算期間1か月超） | 要 | 要 |
| 1年単位の変形労働時間制 | 要 | 要 |
| 1週間単位の非定型的変形労働時間制 | 要 | **不要** |
| 休憩の一斉付与の例外 | **不要** | 不要 |
| 時間外・休日労働（36協定） | 要（**届出しないと免罰効果自体が不発生**） | 要 |
| 代替休暇 | **不要** | 不要 |
| 事業場外労働のみなし（法定超の場合） | 要 | 要 |
| 専門業務型裁量労働 | 要 | 要 |
| 年次有給休暇の時間単位付与 | **不要** | 不要 |
| 年次有給休暇の計画的付与 | **不要** | 不要 |
| 年次有給休暇中の賃金（標準報酬日額） | **不要** | 不要 |

**⚠️ 36協定のみ：届出をしないと免罰効果そのものが不発生**（他は届出前でも効力あり）

### 労使委員会の決議による代替

労使委員会（または労働時間等設定改善委員会）が設置されている場合、
**委員の5分の4以上**の多数による議決により、多くの労使協定を代替できる。

代替できない労使協定：
- **任意貯金**に係る労使協定
- **賃金の一部控除**に係る労使協定
- **年次有給休暇中の賃金**（労働時間等設定改善委員会の決議の場合のみ代替不可）

⚠️ 36協定の代替決議は、**届出することで**効力が生じる（届出が必要）
