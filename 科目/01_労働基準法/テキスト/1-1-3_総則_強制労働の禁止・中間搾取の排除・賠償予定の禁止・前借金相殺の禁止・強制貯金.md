# 1-1-3 総則（強制労働の禁止・中間搾取の排除・賠償予定の禁止・前借金相殺の禁止・強制貯金）

---

## 1. 強制労働の禁止（法5条）★★★★

### 条文
> 使用者は、暴行、脅迫、監禁その他精神又は身体の自由を不当に拘束する手段によって、労働者の意思に反して労働を強制してはならない。

### ポイント

**成立時期**
- 実際に労働させていなくても、「労働を強制したこと」のみで法5条違反が成立する。（昭23.3.2基発381号）
- 労働契約締結時に不当な手段で意思を抑圧した場合も該当。

**暴行があっても違反にならないケース**
- 暴行があっても、**労働強制の目的がなく**「怠けた・態度が悪い」として殴った場合 → 法5条違反にならない（刑法の暴行罪が成立する可能性はある）。

**各手段の定義**（昭63.3.14基発150号）
| 手段 | 定義 | 注意点 |
|------|------|--------|
| 暴行 | 労働者の身体に不法な自然力を行使すること（殴る・蹴る・水をかける等） | 傷害・疼痛は不要 |
| 脅迫 | 本人または親族の生命・身体・自由・名誉・財産に害を加えると通告すること | 積極的言動でなく暗示でも足りる |
| 監禁 | 一定の区画された場所から脱出できない状態に置くこと | **物質的障害は不要**（恐怖により逃げられない状態も該当） |

**「不当」の意味**
- 「不法」に限られない。**たとえ合法的な手段でも**、社会通念上是認し難い程度であれば「不当」に該当する。（昭63.3.14基発150号）

**詐欺との違い**
- 詐欺の手段 → 通常は労働者の意思を無意識に抑圧するものではないため、必ずしも法5条違反にはならない。

**法条競合**
- 法5条違反と刑法の暴行罪・脅迫罪・監禁罪は**吸収関係**（法条競合）。刑法の罪は法5条違反に吸収される。

**罰則（最重い）**
- 1年以上10年以下の拘禁刑 または 20万円以上300万円以下の罰金（法117条）

## 2. 中間搾取の排除（法6条）特★★★★★

### 条文
> 何人も、法律に基づいて許される場合の外、業として他人の就業に介入して利益を得てはならない。

### ポイント

**「何人も」**
- 事業主に限定されない。個人・団体・公人・私人を問わない（公務員も違反行為の主体となりうる）。（昭23.3.2基発381号）

**「法律に基づいて許される場合」**
- 職業安定法による有料職業紹介、許可を受けた委託募集等が該当。

**「利益」**
- 名称・形式を問わない（手数料・報償金・金銭以外の財物、有形無形を問わない）。
- 使用者・労働者・第三者いずれから得た場合も含む。
- **行為者（従業員）が現実に利益を得ていなくても**、法人のために介入行為を行った従業員は処罰される。（昭34.2.16 33基収8770号）

**「業として」**
- 営利目的で**反復継続すること**。1回の行為でも反復継続の意思があれば足りる。主業・副業を問わない。

**労働者派遣との関係**
- 労働者派遣事業（適法・**違法を問わず**）は、そもそも「中間搾取」に該当しない → 法6条違反にならない。（平11.3.31基発168号）

## 3. 賠償予定の禁止（法16条）特★★★★★

### 条文
> 使用者は、労働契約の不履行について違約金を定め、又は損害賠償額を予定する契約をしてはならない。

### ポイント

**禁止内容**
- 違約金または**損害賠償「額」を予定**する契約の禁止。（実際の損害とは無関係にあらかじめ金額を定めること）
- **現実に徴収したときではなく、「契約を締結したとき」に違反が成立**する。（コンメンタール16条）

**禁止されていないこと**
- 現実に生じた損害について賠償を請求すること → 禁止されていない。（昭22.9.13発基17号）

**対象となる損害の種類**
- 労働契約の不履行（有期契約の中途退職・遅刻・無断欠勤等）による損害賠償の予定
- **不法行為**（機械・器具の破損等）による損害賠償の予定も含む。

**契約の相手方**
- 労働者本人だけでなく、**親権者または身元保証人**との契約も禁止。

**趣旨**
- 違約金制度・損害賠償額予定制度が、労働の強制や労働者の意思の不当な拘束につながることを防止するため。

## 4. 前借金相殺の禁止（法17条）特★★★★★

### 条文
> 使用者は、前借金その他労働することを条件とする前貸の債権と賃金を相殺してはならない。

### ポイント

**「前借金」とは**
- 労働契約の締結時またはその後に、**労働することを条件として**使用者から借り入れ、将来の賃金で弁済することを約する金銭。

**禁止内容**
- 「労働することを条件とする」前貸の債権と賃金の**相殺**の禁止。
- 労働者が合意しても禁止（使用者側からの相殺のみ対象）。

**禁止されていないこと**
- 使用者が労働者に金銭を貸すこと自体は禁止されていない。
- 労働者が**自己の意思によって**相殺することは禁止されていない（ただし実質的に使用者の強制と認められる場合は違反）。

**「前借金」に含まれないもの**
- 人的信用に基づく金融（賃金の前払いのような単なる**弁済期の繰上げ**など）で**明らかに身分的拘束を伴わないもの**。（昭33.2.13基発90号）

**生活資金の貸付け**
- 労働組合との労働協約に基づく生活資金の貸付けを賃金から分割控除する場合、**労働することが条件となっていないことが極めて明白な場合**には法17条は適用されない。（昭63.3.14基発150号）

**社会保険料の立替え**
- 休業期間中に使用者が社会保険料の被保険者負担分を立て替え、復職後に賃金から控除する場合は、原則として法17条に抵触しない。（ただし著しい高金利が付される等、労働することが条件と認められる場合は違反）。（平3.12.20基発712号）

**趣旨**
- 金銭貸借関係と労働関係を完全に分離し、金銭貸借関係に基づく**身分的拘束**の発生を防止するため。

## 5. 強制貯金（法18条）特★★★★★

### 条文
> 使用者は、労働契約に附随して貯蓄の契約をさせ、又は貯蓄金を管理する契約をしてはならない。

### 強制貯蓄契約の禁止（1項）

**禁止される強制貯蓄契約の例**
- 労働契約中に貯蓄すべきことが定められている契約
- 貯蓄契約をすることが雇入れの条件となっている契約
- 雇入れ後に貯蓄の契約を締結しなければ解雇する契約

**貯蓄金管理契約の2種類**
| 種類 | 内容 |
|------|------|
| 社内預金 | 使用者自身が直接労働者の預金を受け入れて管理 |
| 通帳保管 | 使用者が銀行等に労働者名義で預け入れ、通帳・印鑑を保管 |

> 「労使協定」を締結しても、強制貯蓄契約はできない。

---

### 任意貯蓄金管理の要件（2〜4項）特★★★★★

任意貯蓄（労働者の委託に基づく）を行う場合の手続：

**① 労使協定の締結 + 所轄労働基準監督署長への届出**（法18条2項、則6条）
- 「労働協約の締結」「就業規則の届出」「労働基準監督署長の許可」ではない。

**② 貯蓄金管理規程の作成・周知**（法18条3項）
- 作業場への備え付け等の措置が必要。
- 貯蓄金管理規程を労働基準監督署長へ届け出る必要はない。

**③ 社内預金の場合：利子の付与義務**（法18条4項）
- 最低利率：**年5厘（年利0.5%）**（預金利率省令4条）
- 上限利率は定められていない。
- 単利・複利を問わない。
- 日歩で定めることもできる。

**社内預金の労使協定に定める事項**（則5条の2）
1. 預金者の範囲
2. 預金者1人あたりの預金額の限度
3. 預金の利率および利子の計算方法
4. 預金の受入れ・払い戻しの手続
5. 預金の保全の方法

> 労使協定の締結事項が規定されているのは「社内預金」のみ。「通帳保管」には定めなし。

---

### 貯蓄金の返還（5〜7項）★

- 労働者が返還を請求したときは、**遅滞なく**返還しなければならない（「◯日以内」ではない）。
- 未返還で**労働者の利益を著しく害すると認められるとき**は、所轄労働基準監督署長が**管理中止命令**を出せる。
- 命令の単位：**労働者ごとに全額**。個々の労働者の貯蓄金の一部についての命令はできない。

## ひっかけポイント一覧

| よく間違えるパターン | 正しい表現 |
|---|---|
| 「実際に労働させてから5条違反が成立」 | **契約時・強制した時点**で成立 |
| 「監禁は物質的障害が必要」 | 物質的障害は**不要** |
| 「不当とは不法なものに限る」 | 合法でも**不当**になりうる |
| 「違法な労働者派遣は中間搾取に該当」 | 適法・違法問わず**該当しない** |
| 「賠償額を徴収したとき16条違反」 | **契約した時点**で違反成立 |
| 「現実の損害賠償請求も禁止」 | 禁止されているのは**金額の予定のみ** |
| 「前借金は全面禁止」 | **相殺**が禁止（貸すこと自体は可） |
| 「労使協定があれば強制貯蓄できる」 | 強制貯蓄は**絶対禁止**（労使協定でも不可） |
| 「任意貯蓄の届出は労働基準監督署長の許可」 | **協定の届出**（許可ではない） |
| 「社内預金の最低利率は年1割」 | **年5厘（0.5%）** |
| 「管理規程を監督署に届け出る必要あり」 | 届出は**不要**（作業場への備え付けのみ） |
