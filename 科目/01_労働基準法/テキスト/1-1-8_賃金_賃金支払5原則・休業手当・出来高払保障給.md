# 1-1-8 賃金の支払（2）賃金支払5原則・非常時払・休業手当・出来高払制の保障給

---

## 賃金支払5原則の概要（法24条）

| 原則 | 例外 | 例外の根拠 |
|------|------|-----------|
| ①通貨払 | あり | 法令・**労働協約**・省令（口座振込等） |
| ②直接払 | なし（使者への支払のみ可） | ― |
| ③全額払 | あり | 法令・**労使協定** |
| ④毎月1回以上払 | あり | 臨時の賃金等 |
| ⑤一定期日払 | あり | 臨時の賃金等 |

> **呪文：「通約全定（つうやくぜんてい）」**
> 通貨払の例外→**労働協約**　全額払の例外→**労使協定**

> **非常時払（法25条）は賃金支払5原則には含まれない。**

---

## ①通貨払の原則

### 例外①：労働協約に別段の定めがある場合
- **労働協約の適用を受ける労働者に限られる**（全労働者に適用されるわけではない）
- 就業規則・労使協定ではNG

### 例外②：省令で定める方法（労働者の「同意」が必要）
- 金融機関の本人名義預貯金口座への振込
- 金融商品取引業者への払込
- **資金移動業者の口座への賃金支払（デジタル払い）**（2023年改正）
  - 上限：**100万円**以下
  - 金融機関口座振込を選択できるようにすること
  - 指定資金移動業者への賃金支払には**労使協定**の締結が必要

> 退職手当のみ、小切手・為替証書による支払も可。

---

## ②直接払の原則

- **例外なし**（ただし「使者」への支払は可）
  - 使者：病気欠勤中の労働者の妻子など、本人の意思を伝達するだけの者
  - 代理人・法定代理人・委任を受けた任意代理人への支払は**違反**

### 重要判例：賃金債権の譲渡（小倉電話局事件・最判昭43）
> 労働者が賃金債権を第三者に譲渡した場合でも、使用者は**直接労働者本人**に支払わなければならない。譲受人は使用者に支払いを求めることができない。

---

## ③全額払の原則

### 例外（控除できる場合）
1. **法令に別段の定めがある場合**（所得税の源泉徴収・社会保険料控除など）
2. **労使協定がある場合**（社宅費用・組合費・社内預金など）

### 端数処理（全額払の原則の例外として認められる）

| 場面 | 切捨て可 | 切上げ可 |
|------|----------|----------|
| 1か月の時間外等の合計時間数 | 30分未満 | 30分以上→1時間 |
| 1時間当たりの賃金額 | 50銭未満 | 50銭以上→1円 |
| 1か月の賃金支払額 | 50円未満 | 50円以上→100円 |
| 1か月の賃金支払額（繰越） | 1,000円未満を翌月に繰越 | ― |

### 重要判例まとめ

| 判例 | 事件名 | 結論 |
|------|--------|------|
| 昭36 | 日本勧業経済会事件 | 不法行為を原因とする損害賠償債権との相殺も**禁止** |
| 昭44 | 福島県教組事件 | 過払賃金の調整的相殺は、合理的接着時期・少額・予告等の要件を満たせば**全額払違反にならない** |
| 平2 | 日新製鋼事件 | 労働者が**自由な意思**に基づき相殺に同意した場合は全額払に違反しない |
| 昭52 | 三晃社事件 | 競業避止違反の退職者の退職金を半額とすることは**合理性あり・有効** |
| 昭48 | シンガー・ソーイング・メシーン事件 | 退職時の賃金債権放棄は、**自由な意思に基づく**ものなら有効 |

### チェック・オフ（組合費の天引き）
- **労使協定**の締結が必要（最判平元・済生会中央病院事件）
- チェック・オフ協定は罰則回避の効力のみ（最判平5・エッソ石油事件）
- 労働協約形式で締結しても、**個々の組合員の委任**がなければ有効にチェック・オフできない

---

## ④⑤毎月1回以上払・一定期日払の原則

### 例外（臨時の賃金等）
- 臨時に支払われる賃金
- 賞与
- 1か月超の期間の精勤手当・勤続手当・奨励加給等

### 「一定の期日」の意味
- 「毎月末日」「毎週土曜日」→ ✅ 合法
- 「毎月15日から20日の間」「毎月第2土曜日」→ ❌ 違法（日が特定されない）

### 年俸制でも毎月払の原則は適用される
（年俸の12分の1ずつを毎月払う必要はないが、毎月払う必要はある）

---

## 非常時払（法25条）

- 支払期日前であっても、**既往の労働**（すでに働いた分）に対する賃金を支払わなければならない
- **未来の労働に対する賃金を支払う義務はない**

### 非常の場合（則9条）

| 対象者 | 事由 |
|--------|------|
| 労働者本人・収入依存者 | 出産、疾病（業務上・外問わず）、災害 |
| 同上 | 結婚、死亡 |
| 同上 | やむを得ない事由による**1週間以上**の帰郷 |

> 「収入によって生計を維持する者」＝親族でなくても同居人でも可。ただし独立の生計を営む親族は不可。

## 休業手当（法26条）

### 基本ルール
```
使用者の責に帰すべき事由 による休業
→ 平均賃金の100分の60以上を支払う（強行規定）
→ 休業手当は「賃金」→ 法24条（支払5原則）適用
→ 「休日」は休業手当不要
→ 一部休業も該当（平均賃金の60%未満なら差額を支払う）
```

### 「使用者の責に帰すべき事由」の範囲
> **民法536条2項の「債権者の責に帰すべき事由」より広い**（ノース・ウエスト航空事件）
> → 使用者側に起因する経営・管理上の障害を含む

| 該当する ✅ | 該当しない ❌ |
|-----------|------------|
| 親工場の経営難による休業 | 天災事変などの不可抗力 |
| 資材不足・設備不備 | 社会通念上正当なロックアウト |
| 解雇予告なし解雇の予告期間 | ストライキ（特別事情がない限り） |
| 新規採用内定者への自宅待機 | 休電による休業 |

### ストライキ不参加者との関係（ノース・ウエスト航空事件）
```
一部の労働者がストライキ → スト不参加者が就労不能

→ 賃金請求権：失う（特別の事情がない限り）
→ 休業手当請求権：失う（ストは使用者側に起因しない）
```
※ただし使用者がスト不参加者を「合理的な範囲を超えて」休業させた場合はその超過部分は使用者の責となる。

### 中間利益との関係（あけぼのタクシー事件 昭62）
```
違法解雇された労働者が解雇期間中に他社で働いて中間利益を得た場合：

平均賃金の60%まで → 中間利益を控除できない（労働者に保障）
60%を超える部分 → 中間利益を控除できる
利益が40%を超える場合 → 賞与など平均賃金算定基礎外の賃金からも全額控除可
```

## 出来高払制の保障給（法27条）

- 出来高払・請負制の労働者には、**労働時間に応じた一定額**を保障しなければならない
- 「1か月につき一定額」では**保障給にならない**（時間給であることが原則）
- 保障給の目安：**平均賃金の100分の60**程度
- 同種業務の労働者間で技量・経験等による差を設けることは**可能**
- 労働者の責による休業：支払不要
- 使用者の責による休業：法26条（休業手当）が適用されるため法27条の適用なし
