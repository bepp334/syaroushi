# 1-1-15 みなし労働時間制（事業場外労働のみなし労働時間制・専門業務型裁量労働制・企画業務型裁量労働制）

---

## 1. みなし労働時間制とは

通常の方法による労働時間の算定ではなく、所定労働時間・当該業務に通常必要とされる時間・労使協定や労使委員会で決議した時間などをもって労働した時間と**みなす**制度。

- あくまで**「時間計算」の特例**であり、実際の労働時間と関係なく一定時間働いたとみなす仕組み

### みなし労働時間制の種類（法38条の2〜4）

| 種類 | 根拠条文 |
|------|---------|
| 事業場外労働のみなし労働時間制 | 法38条の2 |
| 専門業務型裁量労働制 | 法38条の3 |
| 企画業務型裁量労働制 | 法38条の4 |

---

## 2. 共通ルール

### 重要度★★★★ 休憩・深夜業・休日の規定は排除されない

みなし労働時間制が適用される場合でも、以下は通常どおり適用される：

- **休憩**：使用者は必要な休憩をとれるよう措置が必要
- **深夜業**（午後10時以降）：現実に労働した場合は割増賃金が必要。年少者は深夜業禁止も実労働時間で判断
- **休日労働**：休日に業務に従事した場合は割増賃金が必要

（昭和63年1月1日基発1号、平成12年1月1日基発1号）

### 重要度★ 年少者・妊産婦への不適用

みなし労働時間制の規定は**第4章（一般労働者）の労働時間算定**にのみ適用される。
**第6章（年少者）・第6章の2（妊産婦等）の労働時間に関する保護規定**を適用する場合には、みなし労働時間制は使えず、**実労働時間**で判断される。

> 例：みなし制採用会社で妊産婦が「時間外労働させないで」と請求した場合、時間外労働該当性は**実労働時間**で判断する（みなし時間ではない）

## 3. 事業場外労働のみなし労働時間制（法38条の2）

### 重要度特★★★★★ 採用の要件

以下の**両方**を満たす場合に適用される：
1. 労働時間の全部または一部について**事業場外**で業務に従事
2. 使用者の**具体的な指揮監督が及ばず**、労働時間を**算定することが困難**

> 労使協定の締結は**必須ではない**（協定を結べばより精緻な算定が可能になる）

### みなし労働時間制が適用されない場合（昭和63年1月1日基発1号）

- グループで事業場外労働をする場合で、メンバー中に**労働時間管理をする者**がいるとき
- 無線・ポケットベルなどで**随時使用者の指示**を受けながら労働しているとき
- 事業場で訪問先・帰社時刻など**当日業務の具体的指示**を受けた後、指示どおりに従事し事業場に戻る場合

### 重要度● 判例：海外旅行添乗員（阪急トラベルサポート事件・平成26年1月24日最高裁）

あらかじめ定められた旅行日程に沿った旅程管理業務は、業務に関する指示・報告の方法・内容・実施態様等からすると「労働時間を算定し難いとき」に**該当しない**→事業場外みなし制の適用**不可**

### 重要度★ 在宅勤務の適用要件（平成20年7月28日基発0728002号）

以下のすべてを満たす場合に適用される：

1. 起居寝食など私生活を営む**自宅**で業務を行うこと
2. 情報通信機器が、使用者の指示により**常時通信可能な状態におくこととされていない**こと
3. **随時使用者の具体的な指示**に基づいて業務を行っていないこと

### 重要度★ テレワークの適用要件（令和3年3月25日基発0325第2号）

以下の**両方**を満たす場合に適用される：

**①情報通信機器が、使用者の指示により常時通信可能な状態におくこととされていないこと**

| 通信可能な状態に**該当する**（要件を満たさない） | 通信可能な状態に**該当しない**（要件を満たす） |
|--------------------------------------|--------------------------------------|
| 労働者が自分の意思で通信回線を切断できない | 自分の意思で通信回線自体を切断できる |
| 使用者からの指示に即応しなければならない手待ち状態 | 単に回線が接続されているだけで労働者がPCから離れることが自由 |

- 携帯電話を所持しているだけでは「常時通信可能」にならない（応答・折り返しのタイミングを労働者が判断できる場合）

**②随時使用者の具体的な指示に基づいて業務を行っていないこと**

- 業務の目的・目標・期限などの**基本的事項にとどまる**指示はOK
- 作業内容・時間・方法を具体的に特定する指示はNG

### 労働時間の算定方法

| 状況 | みなし時間 |
|------|-----------|
| 労働時間の**全部**が事業場外 | **所定労働時間**労働したものとみなす |
| 労働時間の**一部**が事業場外 | 事業場内の時間を含めて**所定労働時間**とみなす |
| 通常所定を超えることが必要・全部が事業場外 | **通常必要とされる時間**労働したものとみなす |
| 通常所定を超えることが必要・一部が事業場外 | **事業場内時間**＋**通常必要とされる時間** |

### 重要度★★★★★ 労使協定の締結

- 労使協定を締結し「通常必要とされる時間」を定めた場合 → 協定で定める時間がみなし時間となる
- **事業場外部分**についての時間を協定する（事業場内時間は別途把握が必要）

**届出要否：**
- 協定で定める時間が**法定労働時間を超える場合** → 所轄労働基準監督署長に**届出必要**
- 協定で定める時間が**法定労働時間以下** → 届出**不要**
- 有効期間の定めが必要（労働協約による場合を除く）（則24条の2第2項）

## 4. 専門業務型裁量労働制（法38条の3）

### 重要度★★★★ 対象業務（則24条の2の2第2項、令和5年厚労告115号）

| 業務 | 備考 |
|------|------|
| 新商品・新技術の研究開発、人文科学・自然科学の研究 | |
| 情報処理システムの分析または設計 | システムエンジニア |
| 新聞・出版の記事取材・編集、放送番組の取材・編集 | |
| 衣服・室内装飾・工業製品・広告などの新たなデザインの考案 | デザイナー |
| 放送番組・映画制作のプロデューサーまたはディレクター | |
| コピーライター・システムコンサルタント・インテリアコーディネーター・ゲーム用ソフトウェアの創作・証券アナリスト・金融商品の開発 | |
| 大学における教授研究（主として研究に従事するものに限る） | |
| 公認会計士・弁護士・建築士・不動産鑑定士・弁理士・税理士・中小企業診断士 | |
| 2024改正 銀行・証券会社における合併・買収に関する調査・分析・考案・助言（M&Aアドバイザー） | |

> **注意**：医師・歯科医師は対象外。**社会保険労務士も対象外**。
> 数人でプロジェクトチームを組む場合で、チーフの管理の下に業務遂行・時間配分が行われている場合は対象外。

### 重要度特★★★★★ 採用の要件（法38条の3第1項）

**労使協定**により以下を定め、届出が必要：

| # | 締結事項 |
|---|---------|
| 1 | 対象業務 |
| 2 | **1日当たり**の算定労働時間数（「1週間」「1か月」ではない） |
| 3 | 業務遂行の手段・時間配分について使用者が具体的な指示をしないこと |
| 4 | 労働時間の状況に応じた健康・福祉確保措置 |
| 5 | 苦情処理措置 |
| 6 | 2024改正 対象労働者の**同意**を得ること・同意しない者への不利益取扱い禁止 |
| 7 | 2024改正 同意の**撤回**に関する手続 |
| 8 | **有効期間**の定め |
| 9 | 2024改正 労働者ごとの記録を有効期間中＋**5年間（当分の間3年間）保存**すること |

> 就業規則で代えることはできない（労使協定必須）

### 労使協定の届出

- 定める時間の長さにかかわらず**必ず届出が必要**（事業場外みなしとの相違点）
- ただし届出をしなくても36協定のような免罰効果は生じる（免罰効果はある）

### 記録の保存

- 有効期間中＋有効期間満了後**5年間（当分の間3年間）**
- 保存対象：①健康・福祉確保措置の実施状況、②苦情処理措置の実施状況、③同意及びその撤回

## 5. 企画業務型裁量労働制（法38条の4）

### 重要度● 対象業務

**事業の運営に関する事項**についての**企画、立案、調査及び分析の業務**であって：
- 業務の性質上、遂行方法を**大幅に労働者の裁量**に委ねる必要があり
- 遂行手段・時間配分の決定などに関し使用者が**具体的な指示をしない**こととする業務

**該当しない業務（2024改正含む）：**
- 日常的に使用者の具体的指示の下に行われる業務
- 詳細な手順に即して遂行するよう指示されている業務
- 2024改正 使用者から**始業・終業の時刻を指示**されている業務

### 重要度★★ 採用の要件

**労使協定ではなく**「**労使委員会**」の**委員の5分の4以上の多数による議決**が必要（決議+届出）

| # | 決議事項 |
|---|---------|
| 1 | 対象業務 |
| 2 | 対象労働者の範囲 |
| 3 | **1日当たり**の算定労働時間数 |
| 4 | 健康・福祉確保措置 |
| 5 | 苦情処理措置 |
| 6 | 対象労働者の**同意**を得ること・不利益取扱い禁止 |
| 7 | 2024改正 同意の**撤回**に関する手続 |
| 8 | 2024改正 評価制度・賃金制度変更時の労使委員会への**説明義務** |
| 9 | **有効期間**の定め |
| 10 | 2024改正 労働者ごとの記録を有効期間中＋**5年間（当分の間3年間）保存** |

> 対象労働者は対象業務に常態として従事していることが原則

### 重要度★ 決議の届出

- 所轄労働基準監督署長への届出がなければ**効力が発生しない**（36協定と同様）

### 重要度★ 労働者の同意

- 適用される労働者の**個別同意が必須**（専門業務型より厳格）

### 重要度● 安全配慮義務（平成11年労働省告149号）

使用者は、具体的な指示をしないこととなる対象労働者に対しても、**安全配慮義務を免れない**。

### 記録の保存

- 有効期間中＋有効期間満了後**5年間（当分の間3年間）**
- 保存対象：①健康・福祉確保措置の実施状況、②苦情処理措置の実施状況、③同意及びその撤回

### 重要度● 選択注意平18B 定期報告（法38条の4第4項、則24条の2の5）

| タイミング | 報告先 |
|-----------|--------|
| 有効期間の**始期から起算して6か月以内に1回** | 所轄労働基準監督署長 |
| その後**1年以内ごとに1回** | 同上 |

**2024改正ポイント（改正前との比較）：**
- 改正前：「決議が行われた日」から起算、「6か月以内ごとに1回」
- 改正後：「有効期間の**始期**」から起算、初回**6か月**以内、その後**1年**以内ごと

**報告事項：**
1. 対象労働者の労働時間の状況
2. 健康・福祉確保措置の実施状況
3. 2024改正 同意及びその撤回の実施状況

## 6. 労使委員会

### 定義

賃金・労働時間その他当該事業場における労働条件に関する事項を調査審議し、事業主に対し意見を述べることを目的とする委員会（使用者＋労働者代表の構成員からなるもの）

### 重要度★★ 委員の選出

- 委員の**半数**は、過半数労働組合（ない場合は過半数代表者）が**任期を定めて指名**
- 「投票・挙手で選出」ではない（指名）
- 2024改正 指名は監督または管理の地位にある者以外の者について行わなければならない
- 2024改正 使用者の意向に基づくものであってはならない
- 2024改正 使用者は指名された労働者代表委員が事務を円滑に遂行できるよう必要な配慮をする

### 議事録の作成・保存

- 開催の都度議事録を作成
- 開催日（決議が行われた会議は書面の完結の日）から起算して**5年間（当分の間3年間）**保存
- 当該事業場の労働者に対する**周知**が必要

### 2024改正 開催頻度

労使委員会の運営規程に、**6か月以内ごとに1回**開催することを定める（則24条の2の4第4項ニ）

## 7. 重要度別まとめ・比較表

### 3制度の比較

| 項目 | 事業場外みなし | 専門業務型裁量 | 企画業務型裁量 |
|------|------------|-------------|-------------|
| 採用要件 | 要件該当のみ（協定不要） | **労使協定** | **労使委員会の決議**（4/5以上） |
| 届出 | 時間が法定を超える場合のみ | **常に必要** | **常に必要**（不届では効力なし） |
| 労働者の同意 | 不要 | 2024改正 必要 | 必要 |
| みなし時間単位 | 所定/通常必要な時間 | **1日当たり** | **1日当たり** |
| 定期報告 | なし | なし | あり（初回6か月・以後1年ごと） |

### 届出の比較（重要）

| 制度 | 届出が必要か |
|------|-----------|
| 事業場外みなし（協定で法定超） | 必要 |
| 事業場外みなし（協定で法定以下） | **不要** |
| 専門業務型裁量（時間の長さに関わらず） | **必要** |
| 企画業務型裁量（届出なしでは効力発生せず） | **必要** |
