# 1-1-7 賃金の支払（1）賃金の定義・平均賃金

---

## 1-1-1 賃金の定義（法11条）

### 賃金の3要件（重要度特★★★★★）

```
賃金 = ① 名称のいかんを問わず
       ② 労働の対償として
       ③ 使用者が労働者に支払うすべてのもの
```

---

### 賃金とはみなされないもの（重要度特★★★★★）

| 性質 | 例 |
|------|-----|
| **任意的恩恵的給付**（支給条件が不明確な場合） | 退職手当・結婚祝金・死亡弔慰金・災害見舞金 |
| **福利厚生給付** | 住宅貸与・食事供与・無料乗車証 |
| **企業設備・業務費（実費弁償）** | 旅費・役職員交際費・制服・作業備品・器具損料・社用ガソリン代 |

> **ただし！** 労働協約・就業規則・労働契約などで**あらかじめ支給条件が明確**な退職手当・結婚祝金等は「臨時に支払われる賃金」として**賃金に当たる**（法24条2項の臨時の賃金等）。

---

### 賃金とはみなされないもの（個別）

| 項目 | 賃金該当性 | 理由 |
|------|----------|------|
| **チップ** | ❌ 非賃金 | 「使用者が支払う」に当たらない（客が支払う） |
| **解雇予告手当** | ❌ 非賃金 | 労働の対償でなく代償的給付 |
| **休業補償**（法76条） | ❌ 非賃金 | 法定額を超える部分も含め非賃金 |
| **ストック・オプション** | ❌ 非賃金 | 行使時期・金額が労働者の判断に委ねられており労働の対償でない |
| **生命保険料補助金** | ❌ 非賃金 | 労働の対償でなく福利厚生目的の恩恵的給付 |

---

### 賃金とみなされるもの

| 項目 | 賃金該当性 | 理由 |
|------|----------|------|
| **現物給与**（所定賃金の代わりorあらかじめ約束されたもの） | ✅ 賃金 | |
| **通勤手当・通勤定期券** | ✅ 賃金 | |
| **事業主が代わりに負担する社会保険料・所得税等** | ✅ 賃金 | 労働者が法律上当然負うべき義務を免れるため |
| **休業手当**（法26条） | ✅ 賃金 | 生活保障として支給 |

> **現物給与の代金徴収ルール：**
> 実際費用の1/3以下の代金を徴収する場合 → 「代金と1/3の差額部分」は賃金とみなされる

---

### 食事の供与が「賃金でない」条件（3つすべて満たすこと）

1. 食事供与により賃金の減額を伴わないこと
2. 就業規則・労働協約等に定められ明確な労働条件となっていないこと
3. 食事供与による利益の客観的評価額が社会通念上僅少なものと認められること

### 住宅貸与の注意点

- 原則：福利厚生（賃金ではない）
- 例外：住宅を貸与されない者に「均衡手当」を支払っている場合 → 住宅貸与の利益は評価額を限度として賃金

---

## 1-1-2 平均賃金の定義（法12条）

### 平均賃金を使う5場面（「カ・キュウ・ネン・サイ・ゲン」）（重要度★）

| 頭文字 | 場面 |
|--------|------|
| **カ** | 解雇予告手当 |
| **キュウ** | 休業手当 |
| **ネン** | 年次有給休暇の賃金 |
| **サイ** | 災害補償 |
| **ゲン** | 減給の制裁の制限額 |

---

### 原則的算定式（重要度★★★★★）

```
平均賃金 = 算定事由発生日以前3か月間の賃金総額 ÷ その期間の総日数（暦日数）

※ 端数処理：銭未満切捨て
※ 算定事由発生日は含めない（前日からさかのぼる3か月間）
※ 分母は「総日数（暦日数）」であり「総労働日数」ではない
```

**最低保障（日給・時給・出来高払制の場合）：**
```
賃金総額 ÷ 労働した日数 × 60/100
→ これを下回ってはならない
```

---

### 算定事由発生日（重要度特★★★★★）

| 場面 | 算定事由発生日 |
|------|-------------|
| **解雇予告手当** | 労働者に解雇の通告をした日 |
| **休業手当** | 休業日（複数日の場合は最初の日） |
| **年次有給休暇** | 有給休暇を与えた日（複数日の場合は最初の日） |
| **災害補償** | 事故発生日 or 診断によって疾病の発生が確定した日（「死亡した日」ではない） |
| **減給の制裁** | 意思表示が相手方に到達した日（「事由発生日」「決定日」ではない） |

---

### 賃金締切日がある場合（重要度★★★★）

- 算定期間 = **算定事由発生日の「直前の賃金締切日」**以前3か月間
- 賃金締切日当日に事由が発生した場合でも：直前の締切日からさかのぼる3か月
- 賃金ごとに締切日が異なる場合：それぞれの締切日を使って計算

---

### 算定基礎から控除される期間（法12条3項）（重要度★★★★★）

**分母（日数）と分子（賃金）の両方から控除：**

| 控除される期間 | 控除されない期間（注意） |
|-------------|--------------------|
| 業務上の傷病による療養休業 | **通勤災害**による療養休業は控除されない |
| 産前産後休業（法65条） | **年次有給休暇**は控除されない |
| 使用者の責めに帰すべき事由による休業 | **公民権行使**による休業は控除されない |
| 育児休業・介護休業 | **子の看護休暇**は控除されない |
| 試用期間 | |
| ※正当な争議行為による休業期間（通達） | |
| ※組合専従期間（通達） | |

**一部休業の場合：** その日の賃金が休業手当最低基準（60%）を超えるかにかかわらず、その日と賃金の全額を控除

---

### 通勤定期乗車券の取扱い（重要度特★★★★★）

**通勤手当・定期乗車券は算定基礎から控除されない（算入する）。**
6か月定期券であっても、賃金の前払として算入。

---

### 算定基礎となる賃金に含めないもの（法12条4項）（重要度★★★★）

以下は**分子（賃金総額）から控除**する：

1. **臨時に支払われた賃金**（結婚祝金等の臨時賃金）
2. **3か月を超える期間ごとに支払われる賃金**（年2回の賞与等）
3. **通貨以外のもので支払われた賃金で一定範囲に属しないもの**

> 「家族手当」は除外されない（毎月支払われる通常の賃金として算入）。

---

## 1-1-3 平均賃金の特例（法12条6〜8項）

| 場合 | 誰が定めるか |
|------|------------|
| **雇入れ3か月未満** | 雇入れ後の期間とその期間中の賃金で算定（厚生労働大臣ではない） |
| **日日雇い入れられる者** | **厚生労働大臣**の定める金額 |
| **控除期間が3か月以上 / 雇入れ当日** | **都道府県労働局長**が定める |
| **その他算定し得ない場合** | **厚生労働省労働基準局長**が定める |

> 試用期間中に事由が発生した場合：例外的に試用期間中のデータを算定基礎に算入する（通常は控除されるが、控除すると算定不能になるため）。
