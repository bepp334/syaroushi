# 1-1-6 労働契約（3）解雇予告・退職時の証明・金品の返還

---

## 1-1-1 解雇予告（法20条）

### 解雇予告及び解雇予告手当（重要度特★★★★★）選択注意 令04A/平03A/平01A/昭49AB

```
原則：少なくとも30日前に予告
例外：30日前に予告しない → 30日分以上の平均賃金（解雇予告手当）を支払う
組み合わせ可：予告日数分を短縮する代わりに、不足日数分の手当を支払う
```

**法20条の適用対象は「使用者による解雇」のみ。「労働者からの任意退職」には適用なし。**
退職については民法627条1項（申入れから2週間経過で終了）に従う。

---

### 解雇予告期間の計算（重要度★★★●）選択注意 令04A

- 予告した日は算入しない（翌日から計算）
- **30日間は「暦日」で計算**（労働日ではない。休日・休業日があっても延長されない）
- 例：5月31日に解雇の効力を発生させる → **遅くとも5月1日**に予告が必要（中30日間）
- 「大の月（31日）の1日」に予告すれば「その月の末日」に解雇可能

**短縮の計算式：**
不足解雇予告手当 = 30日 − 解雇予告期間（日数）分の平均賃金

具体例：
- 8/31解雇・8/15予告 → 予告期間16日 → 手当は14日分以上必要
- 8/27解雇・8/14予告 → 予告期間13日 → 手当は17日分以上必要

---

### 解雇予告の取消し（重要度★★★★）

- 原則：**取り消すことができない**（民法540条3項）
- 例外：労働者が具体的事情の下に自由な判断によって**同意を与えた場合**には取消し可
- 労働者が同意しない場合：予告期間満了で労働契約終了→「解雇」（自己都合退職にならない）

---

### 解雇予告期間が到来した後に延期した場合（重要度★）

30日前に予告し期限到来後、解雇日を延期してそのまま使用した場合 → **同一条件での労働契約継続とみなされる → 改めて解雇予告等の手続きが必要**

---

### 解雇予告期間中に解雇制限事由が発生した場合（重要度特★★★★★）

```
予告期間中に業務上負傷等が発生
→ 予告期間満了しても解雇できない
→ ただし「予告の効力が停止」したにすぎず、予告自体は無効にならない
→ 解雇制限期間経過とともに解雇の効力が発生する
```

**1日・2日の軽微な負傷でも解雇制限の適用あり。**

---

### 予告なし・手当なしの解雇の効力（重要度★★★★）細谷服装事件（最高裁 昭和35年3月11日）

**相対的無効説（最高裁の立場）：**

```
予告なし・手当なしで即時解雇通告
↓
即時解雇としては無効
↓（使用者が即時解雇を固執する趣旨でない限り）
以下のいずれか早い時点で解雇の効力が発生：
  ① 通知後30日の期間が経過したとき
  ② 通知後に解雇予告手当の支払をしたとき
```

---

### 解雇予告手当の諸規定

| 事項 | 内容 |
|------|------|
| 支払時期 | 即時解雇の場合は**解雇の申渡しと同時**に支払う |
| 賃金該当性 | **賃金ではない**（労働の対償でないため） |
| 5原則の適用 | **適用なし**（通貨払・直接払不要）。ただし通貨で直接払うよう指導 |
| 他債務との相殺 | **不可**（借金とは別個に取り扱う） |
| 時効 | 一般には時効の問題は生じない（解雇の効力発生要件のため） |
| 供託 | 労働者が受領を拒んだ場合、法務局に供託可 |

---

### 解雇予告と休業手当（重要度★）

30日前に予告し、同時に休業を命じた場合 → 平均賃金の60%の休業手当のみでも、予告から30日経過で労働契約は終了。

---

### その他（重要度★）

- **年少者の解雇**：親権者・後見人の承諾は不要
- **組合専従者**：賃金を受けていなくても在籍している限り解雇予告必要
- **再就職の斡旋**：行っても行わなくても解雇予告の義務とは無関係
- **解雇予告の方法**：口頭でも有効（文書が望ましい）

---

## 1-1-2 解雇予告の例外（即時解雇が可能な場合）（法20条1項ただし書・3項）

**即時解雇が可能な2つのケース（いずれも所轄労働基準監督署長の認定が必要）：**

### ① 天災事変その他やむを得ない事由のために事業の継続が不可能となった場合（★★★★★）

1-1-5の解雇制限の「やむを得ない事由」と**基本的に同一の認定基準**。

| 該当 ○ | 非該当 × |
|--------|---------|
| 火災による焼失（事業主の故意・重大な過失を除く） | 税金の滞納処分による事業廃止 |
| 震災による倒壊・類焼 | 金融難（取引先休業等） |
| | 単なる事業の廃止 |

### ② 労働者の責に帰すべき事由に基づいて解雇する場合（★★★★★）選択注意 昭49C

**該当するもの（例）：**
- 極めて軽微なものを除く事業場内での盗取・横領・傷害等（刑法犯）
- 賭博・風紀紊乱等による職場規律を乱す行為
- 採用条件の要素となる経歴の詐称
- 出勤不良・度重なる欠勤・遅刻（注意指導しても改善なし）

**注意：労働者の帰責性が「軽微」なときは解雇予告または手当の支払が必要。**

---

### 解雇予告除外認定の性格（重要度★★★）

- 原則として解雇の意思表示**前**に認定を受けるべき
- ただし、**意思表示後に認定を得た場合でも解雇は有効**
- 解雇の効力発生日 = **意思表示をした日**（認定のあった日ではない）

---

## 1-1-3 解雇予告の適用除外（法21条）（全類型 重要度特★★★★★）

| 類型 | 条件 | 適用除外が解除される（予告が必要になる）タイミング |
|------|------|--------------------------------------------|
| ①日日雇い入れられる者 | - | **1か月**を超えて引き続き使用されるに至った場合 |
| ②2か月以内の期間を定めて使用される者 | - | **所定の期間**を超えて引き続き使用されるに至った場合 |
| ③季節的業務に4か月以内の期間を定めて使用される者 | 季節的業務に限る | **所定の期間**を超えて引き続き使用されるに至った場合 |
| ④試の使用期間中の者 | - | **14日**を超えて引き続き使用されるに至った場合 |

**重要：②は「2か月以内」が正しい（3か月・6か月ではない）。**
**試用期間の長さは就業規則で自由に設定可（14日以内でなくてよい）。14日超えで適用されるだけ。**

---

## 1-1-4 退職時の証明（法22条）

### 退職時証明書（重要度特★★★★★）

- 労働者の退職の場合（自己退職・解雇・期間満了いずれも）、請求があれば**遅滞なく**交付義務
- 退職時証明を求める回数：**制限なし**

**法定記載事項（5つ）：**
1. 使用期間
2. 業務の種類
3. その事業における地位
4. 賃金
5. 退職の事由（**解雇の場合はその理由を含む**）

**注意：「勤務成績」「賞罰」は法定記載事項ではない。**
**注意：労働者の請求しない事項を記入してはならない（法22条3項）。**

---

### 解雇理由証明書（重要度★）

- 解雇予告がされた日から退職の日までの間、労働者が請求可能
- **即時解雇の場合には適用なし**（ただし退職時証明書として解雇理由を請求可能）
- 請求しない事項の記入禁止（同上）

---

### 就業妨害の禁止（法22条4項）

**「通信」が禁止される情報（制限列挙）：** 国籍・信条・社会的身分・労働組合運動に関する事項のみ

**「秘密の記号」の記入禁止：** すべての事項について禁止

> 区別：通信禁止は4つの限定事項のみ、秘密の記号はすべての事項。

---

## 1-1-5 金品の返還（法23条）（重要度特★★★★★）選択注意 昭53ABCDE

```
労働者の死亡または退職の場合
↓ 権利者の請求があったとき
→ 7日以内に賃金を支払い、金品を返還する

争いがある場合 → 異議のない部分を7日以内に支払い・返還
```

**「7日以内」が適用される規定（労基法本則・則のみ）：**
- 法23条（金品の返還）
- 則47条（障害補償・遺族補償及び葬祭料）

**退職手当は例外：** 権利者から請求があっても、就業規則等で定めた支払時期に支払えば足りる（7日以内不要）。

**解雇予告手当：** 法23条の「7日以内に支払うべき金品」には含まれない（解雇の申渡しと同時に支払う別規定による）。

**請求の日から7日以内より先に賃金支払日が到来する場合は、所定の賃金支払日に支払う。**
