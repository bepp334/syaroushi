# 1-1-14 時間外・休日労働（2）割増賃金・代替休暇

---

## 1-1 割増賃金

### 1-1-1 割増賃金の支払義務

#### 法定労働時間を超えて労働させた場合 ★★★★★

使用者が法33条（災害等・公務）または法36条1項（36協定）の規定により法定労働時間を延長して労働させた場合、**割増賃金を支払わなければならない**（法37条1項）

⚠️ 法33条（災害等・公務）によって時間外労働をさせた場合でも、割増賃金の支払義務あり

---

#### 所定労働時間が法定労働時間より短い場合 ★★★★★

- 所定7時間の事業場で8時間まで労働させた場合、超えた1時間（所定外・法定内）→ 割増賃金**不要**（別段の定めがなければ通常賃金）
- **あくまで「法定労働時間を超えた時間」に対してのみ割増賃金が必要**

---

#### 法定休日に労働させた場合 ★★★★★ 選択注意昭51A

- 法33条または法36条1項の規定により法定休日に労働させた場合 → 割増賃金を支払わなければならない
- 週休2日制で1日のみ出勤、1週の労働時間が40時間以内 → 36協定不要・休日労働の割増賃金不要
- 法定休日と法定外休日を明確に分けておくことが望ましいが、法定休日を明示することは義務ではない
- 国民の祝日・創立記念日などは法35条の「休日」に含まれない → 出勤させても1週40時間超にならない限り割増賃金不要

---

#### 深夜業に従事させた場合 ★★★★ 選択注意昭51B

- 深夜業（原則：午後10時〜午前5時）→ 割増賃金を支払わなければならない（法37条4項）

---

#### 黙示の指示による労働時間 ★

- 使用者の明白な指示または客観的に正規の勤務時間内にできないと認められる黙示の指示によって法定労働時間を超えて勤務した場合 → 時間外労働として割増賃金の支払義務あり

---

#### 遅刻があった場合 ★★★

- 遅刻した分だけ終業時刻を繰り下げて労働させる場合、1日の実労働時間が法定時間を超えなければ → 36協定も割増賃金も不要

---

#### 派遣労働者の場合 ★★

- 派遣先が時間外労働等を行わせた場合 → **派遣元**が割増賃金の支払義務を負う
- 違法な時間外労働であっても割増賃金の支払義務は発生する

---

#### 法33条・36条の手続なしで時間外労働させた場合 ★★★★★

- 法33条または法36条の手続を経ずに時間外・休日労働をさせた場合でも → **割増賃金の支払義務を免れない**（小島撚糸事件・最高裁昭和35年）
- 違法な時間外労働の場合は「一層強い理由で」支払義務あり

---

### 1-1-2 割増賃金の算定

#### 割増賃金の計算基礎 ★★★

- 割増賃金の基礎 = **「通常の労働時間または労働日の賃金の計算額」**（平均賃金でも健康保険の標準報酬日額でもない）

---

#### 割増賃金率 ★★★★★

| 種別 | 割増賃金率 | 根拠 |
|------|-----------|------|
| 時間外労働 | **25%以上**（政令で25%と規定） | 法37条1項・政令 |
| 休日労働 | **35%以上**（政令で35%と規定） | 法37条1項・政令 |
| 深夜業 | **25%以上**（法律で直接規定） | 法37条4項 |
| 時間外+深夜 | **50%以上** | 則20条1項 |
| 休日+深夜 | **60%以上** | 則20条1項 |

⚠️ 休日労働の率は「2割5分以上5割以下の範囲内で政令で3割5分」と規定（2割5分以上5割以下の範囲内という枠は維持）
⚠️ 深夜業の割増賃金率は政令ではなく**法律（法37条4項）で直接2割5分以上**と定められている

- 休日労働が8時間を超えても、深夜業に該当しない限り → **35%のみ**（時間外割増は重ねない）
- 割増賃金を支払わない労使合意は無効（法37条は強行規定）

---

#### 時間外労働が翌日に及んだ場合 ★★

- 翌日の**所定労働時間に及んだ**場合 → 翌日所定労働時間の始期まで25%割増で違反なし
- 翌日の**法定休日に及んだ**場合 → 暦日（0:00〜24:00）単位で割増率を適用。0:00から35%以上の休日割増

---

#### 1か月60時間超の時間外労働 ★

- 1か月で60時間を超えた分 → **50%以上**の割増賃金
- ※中小事業主の経過措置は令和5年4月1日以降は廃止

---

### 1-1-3 変形労働時間制と時間外労働

#### 1か月単位変形制の時間外労働 ★★★★★

| 単位 | 時間外労働となる時間 |
|------|-------------------|
| 1日 | 所定労働時間が8時間超の日 → その所定を超えた時間 / それ以外の日 → 8時間超の時間 |
| 1週間 | 所定労働時間が40時間（44時間）超の週 → その所定を超えた時間 / それ以外の週 → 40時間（44時間）超の時間（1.で時間外となる部分除く） |
| 変形期間 | 変形期間の法定労働時間の総枠を超えた時間（1.または2.で時間外となる部分除く） |

- 休日振替の結果、8時間超所定が設定されていない日に8時間超労働 → 超えた分が時間外
- 労働日が増えた週が40時間（44時間）超 → 超えた分が時間外

---

#### フレックスタイム制の時間外労働 ★★

- **清算期間1か月以内** → 清算期間の法定労働時間の総枠を超えた時間
- **清算期間1か月超** → ①清算期間を1か月ごとに区分した各期間における週平均50時間超の部分 ＋ ②清算期間の法定総枠超の時間（①を除く）

---

### 1-1-4 除外賃金 ★★★★★ 選択注意平13C/昭51DE

割増賃金の算定基礎から**除外**される賃金（法37条5項、則21条）：

| 除外賃金 | 備考 |
|---------|------|
| ①家族手当 | 扶養家族がある者に**家族数に応じて**支給されるもの（一律支給は除外されない） |
| ②通勤手当 | |
| ③別居手当 | |
| ④子女教育手当 | |
| ⑤住宅手当 | 住宅費用に応じて支給されるもの（一律定額・全員一律・住宅以外の要素による支給は除外されない） |
| ⑥臨時に支払われた賃金 | 賞与など |
| ⑦1か月を超える期間ごとに支払われる賃金 | |

⚠️ 除外賃金は名称ではなく**実質**で判断
⚠️ 危険作業手当・年俸制の賞与部分（確定支給額）は除外されない → 算定基礎に含める
⚠️ 除外賃金を算入することは使用者の自由（最低基準なので労働者有利な取り扱いはOK）

---

### 1-1-5 割増賃金の計算基礎となる賃金 ★★★

| 賃金形態 | 計算基礎 |
|---------|---------|
| 時間給 | 時間給そのまま |
| 日給 | 日給 ÷ 1日の所定労働時間数 |
| 日給（日によって所定が異なる） | 日給 ÷ 1週間の1日平均所定労働時間数 |
| 週給 | 週給 ÷ 週の所定労働時間数 |
| 月給 | 月給 ÷ 月の所定労働時間数 |
| 月給（月によって異なる） | 月給 ÷ 1年間の1月平均所定労働時間数 |
| 出来高払制 | 賃金総額 ÷ **総労働時間数**（総所定労働時間数ではない） |

⚠️ 時間外・休日労働には25%または35%だけでなく、**100%分（通常賃金）も含めて125%・135%を支払う**必要がある
⚠️ 出来高払制の場合は100%部分が既に含まれているとみなされるため、**加給額（25%・35%部分）のみ追加支払えばよい**

---

### 1-1-6 割増賃金の定額払い制（固定残業代）

#### 有効要件

定額の手当・基本給への組み込みが直ちに違法ではないが、以下の2要件が必要：

| 要件 | 内容 |
|------|------|
| ①判別（区分）可能性 | 通常の労働時間の賃金に当たる部分と割増賃金に当たる部分が判別できること |
| ②対価性 | 当該手当が時間外労働等に対する対価として支払われるものと認められること |

- 判別・対価性を満たしても、割増賃金額が法定額を下回る場合 → **差額の支払義務あり**
- 対価性の判断：契約書等の記載内容、使用者の説明内容、実際の労働時間等を総合考慮

#### 主要判例まとめ

| 判例 | 要点 |
|------|------|
| 高知県観光事件（H6最高裁） | 完全歩合給で通常賃金と割増賃金が判別できない場合 → 別途割増賃金の支払義務あり |
| 医療法人康心会事件（H29最高裁） | 医師への年俸でも割増賃金部分が不明確なら支払ったとは認められない |
| 日本ケミカル事件（H30最高裁） | 手当の対価性は契約書等の記載・説明・勤務状況などを考慮して判断 |
| 国際自動車事件（R2最高裁） | 賃金体系全体における当該手当の位置付け等にも留意して判断 |

### 1-1-7 割増賃金に係る代替休暇

#### 趣旨・仕組み

- 1か月60時間超の時間外労働に対する50%割増のうち、**25%を超える部分**（追加分）を代替休暇で代替できる
- 代替休暇を取得しても、**通常分の25%割増賃金は必ず支払う必要あり**
- 労使協定の締結が必要（**行政官庁への届出は不要**）
- 当該経過措置（中小企業への不適用）は令和5年4月1日以降廃止

---

#### 代替休暇の算定方法

```
代替休暇の時間数 = (1か月の時間外労働時間数 - 60) × 換算率
換算率 = 代替休暇なしの割増率（50%以上）- 代替休暇ありの割増率（25%以上）
       = 50% - 25% = 25%
```

（例）76時間の時間外労働の場合：（76-60）× 25% = 4時間の代替休暇

---

#### 実施の要件 ★

労使協定で定める事項（則19条の2第1項）：

1. 代替休暇の時間数の算定方法
2. 代替休暇の単位（**1日または半日**の一方または両方）
3. 代替休暇を与えることができる期間

⚠️「1日」= 1日の所定労働時間、「半日」= その1/2（厳密でなくてもよいが定義を協定で定める）
⚠️ 代替休暇の取得期間：60時間超となった**月の末日の翌日から2か月以内** ★
⚠️ 代替休暇は**年次有給休暇とは別の休暇**（年休との合算で代替休暇の単位（1日・半日）に達することはOK）
