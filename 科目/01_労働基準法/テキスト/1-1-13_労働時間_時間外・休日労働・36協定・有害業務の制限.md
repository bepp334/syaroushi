# 1-1-13 労働時間・休憩・休日（5）時間外・休日労働・36協定・有害業務の制限

---

## 1-1 時間外・休日労働が認められる3つのルート


| ルート   | 根拠     | 要件                      | 対象                   |
| ----- | ------ | ----------------------- | -------------------- |
| ①災害等  | 法33条1項 | **所轄労基署長の許可**（急迫時は事後届出） | 全事業場                 |
| ②公務   | 法33条3項 | **許可・届出不要**             | 官公署の国家・地方公務員（別表第1除く） |
| ③36協定 | 法36条1項 | **締結＋所轄労基署長への届出**       | 全事業場                 |


---

## 1-1-1 災害等による臨時の必要（法33条1項） ★★★★★

**要件：所轄労働基準監督署長の「許可」**（認定・承認・認可ではない）

**事態急迫で許可を受ける暇がない場合：事後に遅滞なく届け出る**

⚠️「当該年度の終了時まで」に届出ではない→「遅滞なく」
⚠️ 災害等の場合でも**休憩は与える必要あり**
⚠️ 妊産婦が「請求」した場合は時間外・休日・深夜労働をさせることはできない
⚠️ 年少者には時間外・休日労働をさせることができる（深夜業は不可）
⚠️ 許可の主体は**所轄労働基準監督署長**（派遣労働者の場合は**派遣先**の使用者が許可申請）

**代休付与命令（法33条2項）：**
事後届出があった場合、所轄労基署長が「不適当」と認めるときは、相当する**休憩または休日を与えるよう命じることができる**
→ この代休付与命令による休日は使用者の責に帰すべき休業ではないため**休業手当の支払不要**

**「災害等」に該当するもの（令和元年6月7日基発0607第1号）：**

- 地震・津波・風水害・火災などへの対応（事前の対応を含む）
- 突発的な機械・設備の故障の修理・システム障害の復旧
- サーバーへの攻撃によるシステムダウンへの対応 ✅

**「災害等」に該当しないもの：**

- 単なる業務の繁忙・経営上の必要
- 通常予見される部分的な修理・**定期的な保安** ✗

---

## 1-1-2 公務のため臨時の必要（法33条3項） ★

**要件：許可・届出・報告など一切不要**

対象：官公署の事業（別表第1に掲げる事業を除く）に従事する国家公務員・地方公務員

⚠️ 妊産婦が「請求」した場合は時間外・休日・深夜労働をさせることはできない
⚠️ 年少者には時間外・休日労働は可（深夜業は不可）

---

## 1-1-3 36協定（時間外・休日労働協定） ★★★★★

### 締結と届出

**36協定の締結＋所轄労基署長への届出**の両方が必要
→ 届出がない（締結のみ）では免罰効果なし

⚠️「行政官庁の許可」ではなく「締結＋届出」
⚠️ 法33条のような「急迫時の事後届出」規定はない
⚠️ 常時10人未満の事業場でも届出が必要

### 締結事項（法36条2項）

1. 労働者の範囲（業務の種類・労働者数）
2. 対象期間（**1年間に限る**）
3. 時間外・休日労働させることができる場合
4. **1日・1か月・1年**の延長時間または休日の日数
5. 有効期間の定め（労使協定の場合）・起算日・上限遵守の旨 など

⚠️ フレックスタイム制では「1日」の延長時間の協定は**不要**（1か月・1年のみ）
⚠️ 対象期間は事業完了まで1年未満であっても「1年間」とする必要がある

### 時間外労働の限度時間（法36条3・4項）


| 制度                | 1か月      | 1年        |
| ----------------- | -------- | --------- |
| 通常                | **45時間** | **360時間** |
| 1年単位変形制（対象期間3か月超） | **42時間** | **320時間** |


### 特別条項付き36協定（法36条5項） ★★

通常予見できない業務量の大幅増加等に伴い臨時的に限度時間を超える必要がある場合のみ設定可


| 上限       | 時間          | 備考            |
| -------- | ----------- | ------------- |
| 1か月      | **100時間未満** | 休日労働を含む       |
| 1年       | **720時間以内** | 休日労働を**含まない** |
| 45時間超の月数 | **年6か月以内**  |               |


⚠️「100時間以内」ではなく「100時間**未満**」
⚠️ 年720時間には休日労働を含まない（月100時間・2〜6か月平均80時間には含む）

### 上限規制（法36条6項） ★★

36協定の範囲内であっても以下を守らなければならない：

1. **1か月の時間外＋休日労働：100時間未満**
2. **2〜6か月平均の時間外＋休日労働：80時間以内**（任意の2〜6か月すべて）

⚠️ これに違反→6か月以下の拘禁刑または30万円以下の罰金

### 派遣労働者の場合

36協定を締結・届出する義務を負うのは**派遣元**の使用者

### 時間外労働義務の発生要件（日立製作所武蔵工場事件）

36協定の締結・届出だけでは民事上の義務は発生しない
→ **就業規則等に合理的な定め**があって初めて、労働者に時間外労働義務が発生する

### 指針に関する助言・指導

所轄労基署長は36協定をする**使用者・労働組合・過半数代表者**に対して助言・指導ができる
⚠️ 指針に適合しない36協定を「変更」したり「無効」にしたりする権限はない

## 1-1-4 有害業務の制限（法36条6項1号） ★★★★

**36協定を締結した場合であっても、健康上特に有害な業務は1日2時間以内しか時間外労働させることができない**

⚠️「危険な業務」ではなく「健康上**特に有害な**業務」
⚠️ 有害業務であっても**36協定の締結と届出は必要**（不要になるわけではない）

### 健康上特に有害な業務（則18条）

- 多量の高熱物体を取り扱う業務・著しく暑熱な場所の業務
- 多量の低温物体を取り扱う業務・著しく寒冷な場所の業務
- 有害放射線にさらされる業務
- じんあい・粉末を著しく飛散する場所の業務
- **異常気圧下**における業務
- 削岩機・鋲打機などによる**著しい振動**を与える業務
- **重量物の取扱いなど重激なる業務**
- **強烈な騒音**を発する場所の業務
- 鉛・水銀・クロム・有害物の粉じん・蒸気・ガスを発散する場所の業務

⚠️ **深夜業**は含まれない
⚠️ クレーンなどの危険な機械の運転業務は含まれない

### 有害業務の2時間制限の解釈


| 場面           | 取扱い                                 |
| ------------ | ----------------------------------- |
| 通常の労働日       | 法定労働時間8時間＋2時間＝最大**10時間**            |
| 変形制で所定10時間の日 | 10時間＋2時間＝最大**12時間**                 |
| 休日労働         | 最大**10時間**（休日も同じ趣旨）                 |
| 有害業務＋通常業務が同日 | 有害業務単独で法定時間＋2時間以内ならOK（通常業務は協定範囲内で可） |
