# 1-1-11 労働時間・休憩・休日（3）フレックスタイム制・1年単位の変形労働時間制・1週間単位の非定型的変形労働時間制

---

## 1-1 変形労働時間制の比較（全体像）

| 制度 | 条文 | 採用要件 | 届出 | 有効期間 | 特例(44h)適用 |
|------|------|---------|------|---------|-------------|
| 1か月単位の変形 | 法32条の2 | 労使協定または就業規則 | 要（労使協定の場合） | 要（労使協定の場合） | 1か月以内なら適用可 |
| フレックスタイム制（1か月以内） | 法32条の3 | 就業規則＋労使協定 | **不要** | 不要 | 適用可 |
| フレックスタイム制（1か月超） | 法32条の3 | 就業規則＋労使協定 | **要** | **要** | **不可（40時間のみ）** |
| 1年単位の変形 | 法32条の4 | 労使協定 | 要 | 要 | **不可（40時間のみ）** |
| 1週間単位の非定型的変形 | 法32条の5 | 労使協定 | 要 | **不要** | **不可（40時間のみ）** |

⚠️ 育児等への配慮規定：1か月単位・1年単位・1週間単位に適用。**フレックスタイム制は対象外**

## 1-1-1 フレックスタイム制 ★★★★★

### 概要

労働者が日々の始業・終業時刻・労働時間を自ら決める制度。

### 採用の要件（必須！）

**2つとも必要：**
1. **就業規則その他これに準ずるもの**：「始業・終業の時刻を労働者の決定に委ねる」旨を定める
2. **労使協定**：以下の事項を定める

⚠️ 始業・終業の**両方**を委ねる必要あり。一方だけでは不可（平成11年3月31日基発168号）

### 労使協定の締結事項

| 項目 | 備考 |
|------|------|
| 対象労働者の範囲 | |
| 清算期間（3か月以内） | ⚠️「1か月以内」ではなく「3か月以内」 |
| 清算期間の起算日 | |
| 清算期間における総労働時間 | |
| 標準となる1日の労働時間 | 年次有給休暇取得時の賃金算定基礎 |
| コアタイム・フレキシブルタイムの時間帯（任意） | ⚠️設定は**任意**、強制ではない |
| 清算期間が1か月超の場合：有効期間の定め | |

### 届出の要否

- 清算期間が**1か月以内**：届出**不要**
- 清算期間が**1か月超**：届出**必要**（違反：30万円以下の罰金）

⚠️「清算期間の長さにかかわらず届出が必要」は誤り

### 清算期間における総労働時間の総枠

```
総枠 = 週法定労働時間 × 清算期間の暦日数 ÷ 7
```

- 清算期間**1か月以内**：特例事業は44時間の適用あり
- 清算期間**1か月超**：特例事業でも**40時間のみ**（44時間不可）

### 1か月超の場合の月ごとの上限（中間上限）

各月（1か月ごとの区分期間）ごとに**1週平均50時間**を超えてはならない。
→「最終の総枠」と「各月の中間上限」の2つの規制がかかる。

### 完全週休2日制の特例

曜日のめぐりで総労働時間が総枠を超えてしまう不合理を解消するため、労使協定により：
```
1週間当たりの限度 = （清算期間の所定労働日数 × 8時間）÷（暦日数 ÷ 7）
```

### 労働時間の過不足の繰越し

| 状況 | 取扱い |
|------|--------|
| **超過**（実労働 > 総労働時間） | 超過分を次の清算期間に充当 → **全額払違反（NG）** |
| **不足**（実労働 < 総労働時間） | 不足分を次の清算期間の総労働時間に上積み → 総枠内なら**OK** |

⚠️ 超過分の繰越しは法24条（全額払）違反、不足分の翌期上積みは法24条違反ではない

### その他

- 届出義務違反：30万円以下の罰金（法120条1号）
- フレックスタイム制でも使用者に**労働時間把握義務**あり
- 清算期間1か月超の場合、各月の労働時間実績を労働者に通知することが望ましい
- 派遣労働者：派遣元の就業規則・労使協定・労働者派遣契約の3つが必要
- 休憩：一斉付与が必要な場合は**コアタイム中**に設定

## 1-1-2 1年単位の変形労働時間制 ★★★★

### 概要

業務の繁閑に応じて年間の労働時間を効率配分する制度。

### 採用の要件

**労使協定のみ**（就業規則は不要）
→ 労使協定は所轄労働基準監督署長に**届出必要**（違反：30万円以下の罰金）

### 労使協定の締結事項

| 項目 | 備考 |
|------|------|
| 対象労働者の範囲 | |
| 対象期間（1か月超～1年以内） | ⚠️「1年以内」ではなく「1か月を**超え**1年以内」 |
| 対象期間の起算日 | |
| 特定期間 | 繁忙期。複数設定可。対象期間中の相当部分はNG |
| 対象期間における**労働日**および**労働日ごとの労働時間** | 事前特定が必須 |
| 有効期間の定め | 必須 |

⚠️ 「労働日を特定する」＝休日を特定することでもある。「7月〜9月で労働者が指定する3日間を休日」はNG

### 総労働時間の総枠

```
総枠 = 40時間 × 対象期間の暦日数 ÷ 7
```

⚠️ 特例事業でも**40時間**（44時間は不可）

### 労働日数・労働時間・連続労働日数の限度

**労働日数の限度（対象期間が3か月超の場合）**

| 対象期間 | 限度 |
|---------|------|
| 1年 | **280日** |
| 3か月超～1年未満 | 280日 × 暦日数 ÷ 365 |
| 3か月以内 | 限度なし |

**労働時間の限度**

- 1日：**10時間**（タクシー隔日勤務者は**16時間**）
- 1週間：**52時間**
- ⚠️「1日10時間・1週54時間」は誤り

**対象期間が3か月超の場合の追加制限**
- 48時間超の週が**連続して3週以下**
- 3か月ごとの区分期間で48時間超の週の初日が**3以下**

**連続労働日数の限度**

| 期間 | 限度 |
|------|------|
| 対象期間（通常） | **6日** |
| 特定期間 | **12日**（1週に1日の休日確保） |

### 対象期間を1か月以上に区分する場合

- 最初の期間：**労働日と労働日ごとの労働時間**を特定（労使協定時）
- 2期目以降：**労働日数と総労働時間**のみ定め、各期開始の**少なくとも30日前**に過半数労組・代表者の**同意**を得て労働日・各日の労働時間を書面で特定

⚠️ 「個々の労働者の同意」ではなく「過半数労働組合または過半数代表者の同意」
⚠️ 「各期間の開始の日まで」ではなく「少なくとも30日前まで」
⚠️ この書面は所轄労働基準監督署長への届出不要

### 途中退職者への賃金精算

対象期間途中で退職した者など、実際に働いた期間が対象期間より短い場合：
→ 実働期間を平均して**週40時間超**の時間（法33条・36条による時間外を除く）に対して割増賃金を支払う（法24条違反になるため）

⚠️「1日8時間超」に対する精算ではなく「週40時間超」に対する精算

## 1-1-3 1週間単位の非定型的変形労働時間制 ★★★

### 概要

日ごとの繁閑が著しく、かつ定型的に定まらない事業に特化した制度。毎週ごとにシフトを決められる。

### 対象事業場（2要件とも必要）

1. 常時使用する労働者数が**30人未満**
2. **小売業・旅館・料理店・飲食店**のいずれか

⚠️「または」ではなく「かつ」（両方満たす必要あり）

### 採用の要件

- **労使協定**が必要（就業規則は不要だが、民事上の義務には就業規則等の根拠が別途必要）
- 労使協定は所轄労働基準監督署長に**届出必要**
- 有効期間の定め：**不要**

### 1週間ごとの通知

各日の労働時間を**1週間の開始前に書面**で通知する。
緊急・やむを得ない場合は**変更する日の前日まで**に書面で通知すれば変更可。

### 労働時間の限度

- 1日：**10時間**
- 1週間：**40時間以内**（⚠️特例事業でも44時間不可）

## 重要ポイント総まとめ

### 44時間特例が使えるのは？

| 制度 | 44時間特例 |
|------|-----------|
| 1か月単位の変形制 | **使える** |
| フレックスタイム制（1か月以内） | **使える** |
| フレックスタイム制（1か月超） | **使えない** |
| 1年単位の変形制 | **使えない** |
| 1週間単位の非定型的変形制 | **使えない** |

### 届出が不要なのは？

フレックスタイム制（清算期間1か月以内）のみ

### 有効期間の定めが不要なのは？

- フレックスタイム制（清算期間1か月以内）
- 1週間単位の非定型的変形制
