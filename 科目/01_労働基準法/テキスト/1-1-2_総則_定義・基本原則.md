# 労働基準法 1-1-2 総則（定義・基本原則）
出典：スタディング 1-1-2

---

## 重要度マップ
- 特★★★★★：労働者の定義（使用従属性）／使用者の定義／対等の立場／職制上の地位／男女同一賃金「賃金のみ」／公民権行使「給与は自由」
- ★★★★：請負と労働関係／在籍出向／公の職務／公職就任と懲戒解雇
- ★★★●：均等待遇（国籍・信条・社会的身分）／均等待遇「理由として」
- ★★●：公民権行使の保障（選択注意：平20B）

## 1. 労働者の定義（法9条）特★★★★★

### 条文
> 「職業の種類を問わず、事業又は事務所に**使用される**者で、**賃金を支払われる**者」

### 労働者性の判断基準＝使用従属性（2つの柱）

**①指揮監督下の労働**
- 諾否の自由の有無（仕事の依頼・指示を断れるか）
- 業務遂行上の指揮監督の有無（内容・方法・時間を具体的に指示されるか）
- 拘束性の有無（時間・場所が決められているか）
- 代替性の有無（自分の代わりに他の人に任せられるか）→ 代替できると使用従属性が弱まる

**②報酬の労務対償性**
- 時間給基礎の計算、欠勤控除あり、残業手当あり → 使用従属性を補強する

### 労基法と労組法・労契法の「労働者」の違い ⚠️
| 法律 | 定義 | 失業者 |
|------|------|--------|
| **労働基準法（法9条）** | 事業に使用され賃金を支払われる者 | 含まない |
| 労働組合法（法3条） | 賃金・給料その他これに準ずる収入によって生活する者 | **含む** |
| 労働契約法（法2条） | 使用者に使用されて労働し、賃金を支払われる者 | 含まない |

### 具体的類型（労働者に該当するか）
| 類型 | 労働者？ | 根拠・理由 |
|------|--------|-----------|
| 法人の代表取締役 | ✗ | 事業主体との関係で使用従属関係に立たない |
| 業務執行権・代表権なしの重役（工場長・部長等で賃金あり） | ✓ | その限りにおいて労働者（昭和23年3月17日基発461号） |
| 組合専従職員（在籍のまま組合事務に専従） | ✓ | 労働関係は存続（昭和24年6月13日基収1073号） |
| 形式上の請負契約・実体で使用従属関係あり | ✓ | 実体で判断（昭和22年11月27日基発400号）|
| 形式上の請負契約・使用従属関係なし（建物修理の大工など） | ✗ | 請負関係 |
| インターンシップ（見学・体験のみ、指揮命令なし） | ✗ | 使用従属関係なし（平成9年9月18日基発636号） |
| インターンシップ（直接生産活動・使用従属関係あり） | ✓ | 個々の実態で判断 |
| 研修医（病院の指揮監督下で医療行為に従事） | ✓ | 最高裁H17・関西医科大学附属病院事件 |
| 学生（学業課程の実習） | ✗ | 原則として非該当（昭和57年2月19日基発121号） |
| 競輪選手 | ✗ | 昭和25年4月24日基収4080号 |
| 芸能タレント（下記4条件すべて該当） | ✗ | 昭和63年7月30日基収355号 |
| 車持ち込み運転手（最高裁H8・旭紙業事件） | ✗ | 自己の危険と計算・指揮監督弱い |

**芸能タレントが非該当となる4条件（すべて該当する場合のみ）：**
1. 歌唱・演技などが他人によって代替できず、個性が重要
2. 報酬が稼働時間に応じて定められるものでない
3. プロダクションとの関係では時間的に拘束されない
4. 契約形態が雇用契約でない

**車持ち込み運転手（旭紙業事件・最高裁平成8年）のポイント：**
- 自己の危険と計算の下に運送業務に従事
- 運送物品・運送先・納品時刻の指示以外は特段の指揮監督なし
- 時間・場所の拘束は一般の従業員と比べてはるかに緩やか
→ 労基法・労災保険法上の労働者に該当しない

**研修医（関西医科大学附属病院事件・最高裁平成17年）のポイント：**
- 臨床研修は教育的側面を有するが、医療行為等は病院開設者の労務遂行という側面を持つ
- 病院開設者の指揮監督の下に行ったと評価できる → 労働者に当たる

## 2. 使用者の定義（法10条）特★★★★★　選択注意：平21A/昭62DE

### 条文
> 「**事業主**又は**事業の経営担当者**その他その事業の**労働者に関する事項について、事業主のために行為をするすべての者**」

### 各用語の定義
| 用語 | 内容 |
|------|------|
| **事業主** | 個人企業→企業主個人、法人→**法人そのもの**（代表取締役個人ではない） |
| **事業の経営担当者** | 事業経営一般について権限と責任を負う者（法人代表者・支配人など） |
| **労働者に関する事項について事業主のために行為をするすべての者** | 人事・給与・厚生・業務命令・具体的指揮監督を行う者すべて |

### 重要な補足
- 使用者とは実質的に一定の権限を与えられている者 → **係長・課長などの名称ではなく実質で判断**
- 単に上司の命令の**伝達者にすぎない**場合は使用者ではない
- **相対的概念**：労働者であっても、ある事項については使用者になりえる（同一人が両方になることがある）
- 「使用者は企業内で地位の高い者として一律に決まる」わけではない

### 出向・派遣の場合
| 類型 | 使用者 |
|------|--------|
| **在籍型出向** | 出向元・出向先**双方**が責任を負う（三者間の取決めによる権限と責任に応じて） |
| **移籍型出向（転籍）** | **出向先のみ**（出向元との労働契約は終了） |
| **労働者派遣** | 原則**派遣元**が責任。ただし労働者派遣法44条により**派遣先も一部の事項で責任**を負う |

**派遣先が国・地方公共団体の場合でも**、派遣された労働者に対しては労働者派遣法44条が適用される。

### 下請負人
自己の危険と計算の下に、雇用する労働者を使って業務を行い、注文主から独立して処理するものは → **事業主**（労働者ではない）

### 労契法との「使用者」定義の違い ⚠️
| 法律 | 使用者の定義 |
|------|-------------|
| 労働基準法（法10条） | 事業主または経営担当者その他事業主のために行為をするすべての者 |
| 労働契約法（法2条） | **その使用する労働者に対して賃金を支払う者** |

## 3. 労働条件の決定（法2条）特★★★★★　選択注意：平19A/平09B

### 条文
> 1項「労働条件は、**労働者と使用者が、対等の立場において決定**すべきものである。」
> 2項「労働者及び使用者は、**労働協約、就業規則及び労働契約**を遵守し、誠実に各々その義務を履行しなければならない。」

### 重要ポイント
- 「対等の立場」= 形式的だけでなく**実質的**に対等
- 現実に労組があるかどうか・団体交渉で決めたかどうかは**問わない**
- 2項の遵守義務は**労働者・使用者の双方**に課される（使用者のみではない）

### 2項に含まれる3つ・含まれないもの ⚠️
| 含まれる | 含まれない |
|---------|----------|
| 労働協約 | **労使協定**（36協定等）← 免罰効果のみ・規範的効力なし |
| 就業規則 | |
| 労働契約 | |

### 罰則
- **罰則なし**（訓示的規定）
- 法2条違反に対して労基監督機関が職権を行使することも**原則として行われない**
- 「使用者のみ罰則」ではなく、**双方ともに罰則なし**

### 各用語の定義
| 用語 | 定義 |
|------|------|
| 労働協約 | 労働組合と使用者（またはその団体）が労働条件等について合意した協定。書面+両当事者の署名または記名押印で効力発生 |
| 就業規則 | 労働者の就業上遵守すべき規律・労働条件に関する具体的事項を使用者が定めた規則 |
| 労働契約 | 個々の労働者が使用者から対価を得て、当該使用者の下で自己の労働力の処分を委ねることを約する契約 |

## 4. 均等待遇（法3条）★★★●　選択注意：昭61BC

### 条文
> 「使用者は、労働者の**国籍、信条又は社会的身分**を理由として、賃金、労働時間その他の**労働条件**について、差別的取扱をしてはならない。」

### 各用語の定義
| 用語 | 定義 |
|------|------|
| **国籍** | 国民たる資格。二重国籍者・無国籍者も含む |
| **信条** | 特定の宗教的または政治的信念（昭和22年9月13日発基17号） |
| **社会的身分** | 生来的な地位（あとから自分の努力で変えられない社会的な位置）。憲法14条の「門地」も含む |

### 重要な限定 ⚠️
- **「性別」は含まれない**（憲法14条との混同注意！）→ 性差別は**男女雇用機会均等法**で規制
- **限定列挙**：この3つ以外の理由による差別は3条違反にならない
- 「差別的取扱い」= 有利または不利に取り扱うこと（不利のみではない）
- 法3条の「労働条件」= 解雇・賃金・労働時間・災害補償・安全衛生・寄宿舎など職場の待遇の一切

### 適用範囲（雇入れは対象外）
- 法3条は「**雇入れ後**」の労働条件の制限であって、**雇入れそのものを制約する規定ではない**（三菱樹脂事件・最高裁S48）
- 思想・信条を理由とする雇入れ拒否は直ちに違法とはならない
- 採否決定に際し思想・信条の調査や申告を求めることも法律上禁止された違法行為ではない

### 職制上の地位の扱い 特★★★★★
- 「臨時工と常用工」「工員と職員」など職制上の地位に基づく待遇差異 → **差別的取扱いに含まれない**
- 特定の信条に従う行動が**企業秩序維持に重大な影響を及ぼす場合**、その秩序違反行為を理由とした取扱いは3条違反にならない

## 5. 男女同一賃金の原則（法4条）特★★★★★（趣旨）

### 条文
> 「使用者は、労働者が**女性であること**を理由として、**賃金**について、男性と差別的取扱いをしてはならない。」

### 重要ポイント
- **「賃金についてのみ」**の規定 → 賃金以外の労働条件の性差別は4条違反とはならない（均等法で規制）
- 「女性であることを理由として」= 以下を理由にすることがすべてNG：
  - 女性であること自体
  - 女性は一般的に能率が悪いこと
  - 女性は勤続年数が短いこと
  - 女性は主たる生計の維持者ではないこと
  - **その事業場で実際に女性の平均勤続年数が短い事実があっても**→NG
- 「差別的取扱い」= **不利**だけでなく**有利**に取り扱う場合も含む

### 差別待遇を定める就業規則の扱い
- 就業規則に男女差別的な賃金規定があっても、**現実に差別事実がない**場合 → その規定は**無効だが4条違反にならない**
- 現実に差別待遇の事実がある場合 → 4条違反（過去問R04-04C）

### 合法な個人差
- 職務・能率・技能・年齢・勤続年数などによる賃金の個人的差異 → 4条違反ではない

## 6. 公民権行使の保障（法7条）★★●　選択注意：平20B

### 条文
> 「使用者は、労働者が労働時間中に、**選挙権その他公民としての権利を行使**し、又は**公の職務を執行**するために**必要な時間を請求**した場合においては、**拒んではならない**。ただし、権利の行使又は公の職務の執行に妨げがない限り、**請求された時刻を変更**することができる。」

### 重要ポイント
- 「拒む」ことを禁止 → 使用者が拒んだだけで**本条違反が成立**
- **時刻の変更は可**（妨げがない限り）。「日時の変更」ではなく「時刻のみ」変更可
- 就業規則に「公民権行使の労働時間外実施」を定めても、それを理由に就業時間中の請求を**拒否すれば違法**
- **給与は有給・無給とも当事者の自由**（法7条には給与に関する規定なし）特★★★★★

### 公民としての権利
| 該当する | 該当しない |
|---------|----------|
| 公職の選挙権・被選挙権 | 個人としての訴権の行使 |
| 最高裁判所裁判官の国民審査 | **他の候補者のための選挙運動** |
| 地方自治法による住民の直接請求 | |
| 行政事件訴訟法による民衆訴訟 | |
| 選挙人名簿に関する訴訟・選挙に関する訴訟 | |

### 公の職務
| 該当する | 根拠 |
|---------|------|
| 衆議院議員等の議員の職務 | 公務に民意を反映する職務 |
| 検察審査員・労働審判員・裁判員の職務 | 公務に民意を反映する職務 |
| 民事訴訟法による証人の職務 | 公務の公正妥当な執行 |
| 公職選挙法による投票立会人 | 公務の適正な執行を監視 |
| 非常勤の消防団員の職務 | |
| 予備自衛官の防衛招集・訓練招集 | |

### 公職就任と懲戒解雇（判例）特★★★★
- 公職就任を**使用者の承認にかからしめ**、無承認就任者を**懲戒解雇**にする就業規則条項 → **無効**（十和田観光電鉄事件・最高裁S38）
- 当該公職就任が業務遂行を著しく阻害するおそれがある場合でも → 懲戒解雇は無効
- **普通解雇はOK**（懲戒解雇のみ無効）

## ひっかけまとめ
| よく間違えるパターン | 正しい表現 |
|---|---|
| 法3条に「性別」が含まれる | **含まれない**（均等法で規制） |
| 法2条2項に「労使協定」が含まれる | **含まれない**（3つのみ） |
| 在籍出向→出向元のみ責任 | 出向元・出向先**双方**が責任 |
| 形式上の請負→常に非労働者 | **実体**で使用従属関係があれば労働者 |
| 事業主＝代表取締役 | 法人の場合は**法人そのもの** |
| 「管理監督者」が使用者の定義に出てくる | **「経営担当者」**が正しい（管理監督者は41条） |
| 公民権行使→有給にしなければならない | 有給・無給は**当事者の自由** |
| 業務支障があれば公民権行使を拒める | **拒んではならない**（時刻変更のみ可） |
| 公職就任して業務阻害→懲戒解雇OK | 懲戒解雇は**無効**。普通解雇はOK |
| 女性の平均勤続年数が短い事実→賃金差別OK | それでも「女性であることを理由として」でNG |
| 労組法の労働者≒労基法の労働者 | 労組法は**失業者を含む**（労基法は含まない） |
