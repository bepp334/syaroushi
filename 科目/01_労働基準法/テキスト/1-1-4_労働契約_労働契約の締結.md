# 1-1-4 労働契約（労働契約の締結）

---

## 1. 労働基準法違反の契約（法13条）特★★★★★

### 条文
> 労働基準法で定める基準に達しない労働条件を定める労働契約は、その部分については無効とする。この場合において、無効となった部分は、労働基準法で定める基準による。

### ポイント

**2つの効力**
| 効力 | 内容 |
|------|------|
| **強行的効力** | 基準に達しない部分を「無効」にする |
| **直律的効力** | 無効となった部分を「労働基準法の基準で補充」する |

- 「基準に達しない部分のみ」が無効 → **有利な部分まで無効になるわけではない**
- 労働契約自体は有効に継続する

**関連比較**
| 根拠 | 基準 | 表現 |
|------|------|------|
| 労働基準法13条 | 労基法の基準 | 「達しない」部分を無効 |
| 労働組合法16条 | 労働協約の基準 | 「違反する」部分を無効 |
| 労働契約法12条 | 就業規則の基準 | 「達しない」部分を無効 |

**採用内定（大日本印刷事件 昭54.7.20）**
- 採用内定通知＋誓約書の提出 → **「就労始期付・解約権留保付労働契約」が成立**
- 内定取消しが許されるのは：「採用内定当時知ることができず、知ることが期待できないような事実」で「客観的に合理的・社会通念上相当」な場合に限る

**試用雇用契約（神戸弘陵学園事件 平2.6.5）**
- 雇用期間を設けた趣旨・目的が「労働者の適性評価のため」であるときは、当然終了の明確な合意がない限り、期間は「契約の存続期間」ではなく「試用期間」と解する
- 試用期間中の解雇は、通常の解雇より広い範囲で許されるが、客観的合理的理由・社会通念上の相当性が必要

## 2. 契約期間（法14条）特★★★★★

### 条文（1項）
> 労働契約は、期間の定めのないものを除き、一定の事業の完了に必要な期間を定めるもののほかは、3年（一定の場合にあっては5年）を超える期間について締結してはならない。

### 有期労働契約の上限

| 種類 | 上限 |
|------|------|
| 原則 | **3年** |
| 高度の専門的知識等を有する労働者（当該業務に就く者） | **5年** |
| 満60歳以上の労働者 | **5年** |
| **一定の事業の完了に必要な期間を定めるもの** | **上限なし** |
| 期間の定めのない労働契約 | 適用除外（解約の自由あり） |

**「一定の事業の完了に必要な期間」の要件**
- 有期的事業であることが客観的に明らかな場合
- その事業の終期までの期間を定める契約であること
- 例：5年のダム工事に技師を5年間の契約で雇い入れる

### 高度の専門的知識等（5年上限の適用）

高度の専門的知識等を有する労働者でも、**当該高度の専門的知識等を必要とする業務に就かない場合は3年が上限**。

**対象者（厚生労働大臣が告示）**
1. 博士の学位を有する者
2. 公認会計士・医師・歯科医師・獣医師・弁護士・1級建築士・税理士・薬剤師・**社会保険労務士**・不動産鑑定士・技術士・弁理士
3. ITストラテジスト・システムアナリスト・アクチュアリー試験合格者
4. 特許発明の発明者・登録意匠の創作者・登録品種の育成者
5. 年収**1,075万円**以上の技術者・システムエンジニア等

> 「1,075万円」は暗記必須。高度プロフェッショナル制度・無期転換ルールの特例にも登場する。

### 上限を超えた場合の効果

- 労働契約**自体は有効**（無効にはならない）
- 「期間の定めのない契約」にもならない
- 超過部分が**上限（3年または5年）に修正**される

### 有期労働契約の更新

- 上限内で更新することにより、結果としてトータルが上限を超えても**適法**
- 更新期間にも契約期間の上限が適用される（1回の更新期間が上限を超えてはならない）

## 3. 有期労働契約の特例（附則137条）★★★

### 条文
> 1年を超える期間の定めのある労働契約（一定の事業の完了に必要な期間を定めるものを除き、5年上限対象者を除く）を締結した労働者は、期間の初日から**1年を経過した日以後**は、使用者に申し出ることにより、**いつでも退職**できる。

### 対象・除外

| | 内容 |
|---|---|
| 対象 | 1年超の有期労働契約を締結した労働者 |
| **除外①** | 一定の事業の完了に必要な期間を定めた労働者 |
| **除外②** | 5年上限対象者（高度専門・満60歳以上） |

- 1年を経過した日以後に「申出」すれば退職可
- 民法628条（やむを得ない事由のない期間中の解約禁止）の例外

## 4. 雇止めの予告・理由の明示（令5厚労告114号）★★★

### 雇止めの予告が必要なケース
次のいずれかに該当する有期労働契約を更新しない場合は、**少なくとも契約満了日の30日前まで**に予告が必要（ただし、あらかじめ更新しない旨が明示されているものを除く）：

1. **3回以上更新**した契約
2. **雇入れの日から1年を超えて継続勤務**している者に係る契約

### 雇止めの理由の明示
- 予告後または雇止め後に労働者から「理由の証明書」を請求されたら → **遅滞なく交付**

### 契約期間についての配慮
- 1回以上更新かつ1年超継続勤務の者の契約を更新する場合 → **契約期間をできる限り長くするよう努力義務**

## 5. 労働条件の明示（法15条）特★★★★★

### 条文（1項）
> 使用者は、労働契約の締結に際し、労働者に対して賃金、労働時間その他の労働条件を明示しなければならない。賃金・労働時間その他厚生労働省令で定める事項については、原則として書面の交付により明示しなければならない。

### 明示の時期
- **「労働契約の締結の際」**（募集時点は不要。ただし職業安定法上の義務はある）
- 有期契約の**更新のたびに**・再雇用のたびに最新の労働条件を明示

### 絶対的明示事項（書面等による明示が必要）

| 番号 | 事項 | 書面必須？ |
|------|------|-----------|
| 1 | 労働契約の期間 | ✅ |
| 2 | 有期労働契約を更新する場合の基準（通算期間・更新回数の上限含む） | ✅（更新可能性がある契約のみ） |
| 3 | 就業の場所・従事すべき業務（**変更の範囲を含む**） | ✅ |
| 4 | 始業・終業時刻、時間外労働の有無、休憩・休日・休暇・就業時転換 | ✅ |
| 5 | 賃金の決定・計算・支払方法、締切日・支払時期、**昇給** | ✅（**昇給以外**） |
| 6 | 退職（解雇事由含む） | ✅ |

> **「昇給に関する事項」のみ書面不要**（定めの有無・内容が確定的でないため）

**書面以外の方法**：労働者が希望する場合はFAX・電子メール等でも可

### 相対的明示事項（定めがある場合に明示義務。口頭でも可）

退職手当・賞与・食費・安全衛生・職業訓練・災害補償・表彰制裁・休職

### 2024年改正：無期転換申込権が発生する契約の締結時の追加明示

無期転換申込権が発生する有期契約の締結時には、通常の明示に加えて：
1. 無期転換申込みに関する事項（申込権があること）→ **書面等必須**
2. 無期転換後の労働条件 → **書面等必須**（昇給・更新基準を除く絶対的明示事項）

また、無期転換後の賃金等の決定に当たり、正社員等とのバランスを考慮した事項について**説明するよう努力義務**あり。

### 派遣労働者
- 明示義務は**派遣元**のみ（派遣先には課されない）
- 派遣元は自己が労基法上の義務を負わない労働時間・休憩・休日の事項も含めて明示が必要

### 比較：健康保険・厚生年金・雇用保険の適用に関する事項
- **職業安定法（求人時）** → 明示事項に含まれる
- **労働基準法（労働契約締結時）** → 明示事項に**含まれない**

## 6. 即時解除と帰郷旅費（法15条2項・3項）★★★★

### 即時解除（2項）
- **明示された労働条件と事実が相違**する場合 → 労働者は**即時に**労働契約を解除できる
- 「即時」＝予告なし
- 解除の効力：**将来に向かって**消滅（遡及効なし。民法と異なる）

**解除できる「相違」の範囲**
- 法15条1項で**明示すべきこととされている労働条件の相違**に限る（すべての明示事項ではなく、法定明示事項の相違）
- 「単なる福利厚生施設とみなされる社宅」の不提供 → 解除権の対象外（賃金に該当する場合は対象）
- 自己以外の者の労働条件の相違 → 解除権の対象外

### 帰郷旅費（3項）
- 就業のために住居を変更した労働者が、**契約解除の日から14日以内**に帰郷する場合 → 使用者が**必要な旅費を負担**
- 「必要な旅費」：労働者本人だけでなく、労働者により生計を維持されている**同居の親族（内縁の配偶者含む）の旅費**も含む

**罰則**
- 1項（明示義務）・3項（旅費負担） → 罰則あり
- 2項（即時解除権）→ 罰則なし（労働者の権利を与える規定のため）

## ひっかけポイント一覧

| よく間違えるパターン | 正しい表現 |
|---|---|
| 「基準に達しない労働条件→労働契約全体が無効」 | **その部分のみ**無効 |
| 「上限超えの有期契約は無効 or 期間の定めなし」 | 契約は有効、**上限期間に修正**される |
| 「60歳以上は65歳まで契約可能」 | 60歳以上は**5年以内**（65歳という上限ではない） |
| 「高度専門なら業務に関係なく5年」 | **当該業務に就く場合のみ**5年（それ以外は3年） |
| 「昇給は書面で明示必須」 | 昇給は**書面不要**（絶対的明示事項だが書面不要な例外） |
| 「募集時点で労働条件を明示すべき」 | **労働契約の締結の際**に明示（募集時は労基法上は不要） |
| 「健康保険・雇用保険の適用は労基法の明示事項」 | 労基法上は**明示事項に含まれない**（職業安定法は含まれる） |
| 「即時解除で契約は遡及して無効」 | **将来に向かって**消滅（遡及効なし） |
| 「帰郷旅費は本人分のみ」 | **同居の親族（内縁含む）分も**含む |
| 「法15条全項目に罰則あり」 | **2項（即時解除）には罰則なし** |
