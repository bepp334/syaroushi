## 平日1時間の使い方（鉄則）

```
10分：復習キューから昨日の間違い問題を解く
15分：テキスト1節を流し読み
25分：AIに問題4問作成してもらい解く
10分：間違い記録・日次記録
```

## 休日の使い方（2〜3時間）

```
30分：週間復習（間違いデータベースから復習問題）
60分：テキスト2節 + 問題演習
30分：弱点論点の解説を読む
（余力があれば次の節も進める）
```

## 毎日の学習ルーティン（厳守）

### 平日（3時間）
| 時間 | 内容 |
|------|------|
| 20分 | **復習**：復習キュー.md から昨日・先週の間違い問題を解く |
| 60分 | **テキスト**：今日の範囲1節をコピペしてAIに貼り付け、理解する |
| 50分 | **問題演習**：AIに「問題を作って」→ 解く |
| 20分 | **記録**：間違い記録・日次記録を更新 |
| 10分 | **明日の準備**：次の範囲を確認 |

### 土日（5時間/日）
| 時間 | 内容 |
|------|------|
| 90分 | **週間復習**：AIに「今週の間違いから復習問題を作って」→ 解く |
| 120分 | **新範囲**：テキスト + 問題演習 |
| 60分 | **弱点深掘り**：間違いが3回以上の論点を徹底理解 |
| 30分 | **週次振り返り**：週間計画の達成チェック・来週計画の作成 |

## 科目別優先度と学習のコツ

### 最重要科目（配点大・落としてはいけない）
- **国民年金法**（択一10問・選択3問）← 今学習中
- **厚生年金保険法**（択一10問・選択3問）
- **健康保険法**（択一10問・選択3問）

### 選択式の足切りに要注意
- **労一・社一**（一般常識）← 予想外の論点が出る。毎日1問は触れる
- **全科目**：選択式は各3点以上が必要（2点だと選択足切り）

### 数字・期間の暗記が必要な科目
- 労基法：割増賃金率、有給日数
- 雇用保険法：受給日数、各手当の額
- 年金法：保険料、年金額、支給開始年齢
